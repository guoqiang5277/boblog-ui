    表单布局 (Form Layout)

    表单组 (.boblog-form-group)

    • 容器间距: margin-bottom 15px (--boblog-spacing-md)，最后一个 margin-bottom 0
    • 标签 (.boblog-form-label): display block，font-weight bold，字号 12px (--boblog-font-size-sm)，颜色 #333 (--boblog-text-dark)，margin-bottom 5px (--boblog-spacing-xs)
    • 帮助文本 (.boblog-form-help): 字号 11px，颜色 #666 (--boblog-text-gray)，margin-top 5px (--boblog-spacing-xs)
    • 错误提示 (.boblog-form-error): 字号 11px，颜色 #CC3333 (--boblog-red-error)，margin-top 5px (--boblog-spacing-xs)
    • 必填标记 (.boblog-form-required): 伪元素 ::after 添加红色星号 *
    • 本组件为纯 CSS 实现，无需引入 JavaScript
    用户名长度为 4-16 个字符
    密码不能为空
    <!-- 带帮助文本 -->
<div class="boblog-form-group">
    <label class="boblog-form-label">用户名</label>
    <input type="text" class="boblog-input" placeholder="请输入用户名">
    <div class="boblog-form-help">用户名长度为 4-16 个字符</div>
</div>

<!-- 带错误提示 + 必填标记 -->
<div class="boblog-form-group">
    <label class="boblog-form-label boblog-form-required">密码</label>
    <div class="boblog-input-password">
        <input type="password" class="boblog-input boblog-input-error" placeholder="请输入密码">
    </div>
    <div class="boblog-form-error">密码不能为空</div>
</div>

    垂直布局 (.boblog-form)

    • 默认布局方式，标签在控件上方

    • 各表单组垂直堆叠，间距 15px (--boblog-spacing-md)

    • 标签宽度固定 120px，右对齐，padding-top 8px 与输入框文字对齐
    • 标签与控件间距 10px (--boblog-spacing-sm)
    • 适合中等规模的表单

    • padding-top 15px (--boblog-spacing-md)，margin-top 20px (--boblog-spacing-lg)
    • 顶部分隔线 1px solid (--boblog-blue-border-list)
