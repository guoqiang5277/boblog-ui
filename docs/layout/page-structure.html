    页面结构 (Page Structure)

    前台页面结构 (.boblog-page)

    • 页面容器: .boblog-page，min-height 100vh，flex 纵向布局
    • 顶部区域 (.boblog-page-header): border-bottom 10px solid #BEE6FD，padding 15px 0
    • 主体区域 (.boblog-page-body): display flex，flex: 1，padding 20px 0，gap 20px
    • 主内容 (.boblog-page-main): flex: 1，min-width 0
    • 侧边栏 (.boblog-page-sidebar): width 200px (--boblog-sidebar-width)，flex-shrink 0
    • 底部区域 (.boblog-page-footer): border-top 10px solid #BEE6FD，padding 20px 0，居中 12px 文字
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    后台管理布局 (.boblog-admin-page)

    • 页面容器: .boblog-admin-page，min-height 100vh，背景 #f5f5f5
    • 顶栏 (.boblog-admin-topbar): position fixed，height 50px，背景 --boblog-blue-primary，白色文字，z-index 1000
    • 内容区域 (.boblog-admin-body): display flex，padding-top 50px（为固定顶栏留空）
    • 侧边栏 (.boblog-admin-sidebar): width 240px (--boblog-admin-sidebar-width)，position fixed，白色背景，右侧蓝色边框
    • 侧边栏折叠 (.boblog-admin-sidebar.collapsed): width 0，overflow hidden
    • 侧栏折叠手柄 (.boblog-sidebar-handle): position fixed，桌面端（≥992px）显示，跟随侧栏位置
    • 移动端遮罩 (.boblog-sidebar-backdrop): 移动端（≤991.98px），点击关闭侧栏
    • 侧栏树形内边距: .boblog-admin-sidebar .boblog-tree 自动添加 padding
    • 主内容区 (.boblog-admin-main): flex: 1，padding 20px，margin-left 240px
    • 本组件为纯 CSS 实现，侧边栏折叠需要 JS 切换 .collapsed 类

    侧栏辅助组件

    侧栏折叠手柄 (.boblog-sidebar-handle)

    • 定位: position fixed，紧贴侧栏右边缘
    • 显示条件: 仅桌面端（≥ 992px）显示
    • 跟随侧栏: 侧栏折叠时（.sidebar-collapsed），手柄移动到 left: 0
    • 内部按钮: .handle-button，24×36px，居中显示折叠图标（如 « / »）
    • 过渡动画: left 属性使用 0.3s ease 过渡

    移动端遮罩层 (.boblog-sidebar-backdrop)

    • 显示条件: 仅移动端（≤ 991.98px）生效
    • 功能: 侧栏展开时显示半透明遮罩（rgba(0,0,0,0.3)），点击关闭侧栏
    • 激活方式: JS 切换 .active 类控制显示/隐藏
    • 覆盖范围: position fixed，top: 50px（避开顶栏），width/height 100%
    • 层级: z-index 1039（低于侧栏 z-index 1040）

    侧栏树形内边距

    • 选择器: .boblog-admin-sidebar .boblog-tree
    • 效果: padding: 10px 0 10px 10px，为侧栏中的树形菜单提供合适的内边距

    重启遮罩 (.boblog-restart-overlay)

    • 用途: 后台管理系统触发服务重启时的全屏遮罩层
    • 默认状态: display: none（隐藏）
    • 激活方式: JS 添加 .show 类，切换为 display: flex
    • 定位: position fixed，覆盖整个视口，z-index 9999
    • 背景: rgba(0, 0, 0, 0.8) 半透明黑色
    • 内部 spinner: .boblog-restart-overlay .boblog-spinner 固定尺寸 3rem × 3rem
    • 布局: flex 居中，内容区域使用白色文字

    使用说明

    适用场景

    • 前台布局：经典 Bo-Blog 双栏博客页面（主内容 + 侧边栏）
    • 后台布局：管理系统三区域布局（顶栏 + 侧边菜单 + 主内容）
    • 公共元素：页面标题、工具栏、内容区块可在前后台通用

    设计要点

    • 前台使用 Flexbox 纵向布局，header/footer 固定，body 撑满剩余空间
    • 前台顶/底部使用 10px 蓝色粗边框 (#BEE6FD)，还原 Bo-Blog 经典风格
    • 后台顶栏 position fixed，侧边栏 position fixed，主内容通过 margin-left 避让
    • 后台侧边栏支持 .collapsed 折叠，配合 CSS transition 实现动画

    复合布局类

    后台管理常用布局组合的语义化封装，一个类名即可替代多个组合类。

    类名 等效组合 使用场景
    .boblog-panel-footer-bar panel-footer + flex + align-center + justify-between 面板底栏（分页、操作按钮等）
    .boblog-toolbar page-toolbar + py-3 + border-bottom + mb-4 页面工具栏（标题 + 操作按钮）
    .boblog-filter-bar flex + align-center + mb-3 + flex-nowrap 筛选栏（筛选条件 + 按钮）
