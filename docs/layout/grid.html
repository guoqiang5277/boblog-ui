    网格系统 (Grid)

    容器 (.boblog-container)

    • 固定宽度容器: .boblog-container，max-width 980px，水平居中，左右 padding 15px
    • 流式容器: .boblog-container-fluid，width 100%，左右 padding 15px
    • 本组件为纯 CSS 实现，无需引入 JavaScript
    .boblog-container（max-width: 980px）
    .boblog-container-fluid（width: 100%）

    12 列网格 (.boblog-row / .boblog-col-*)

    • 行容器: .boblog-row，display flex + flex-wrap wrap，左右负 margin -7.5px
    • 列容器: .boblog-col-1 至 .boblog-col-12，左右 padding 7.5px（列间距合计 15px）
    • 自适应列: .boblog-col，flex: 1 0 0%，均分剩余空间
    • 自动宽度列: .boblog-col-auto，flex: 0 0 auto，按内容宽度
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    使用说明

    适用场景

    • 博客页面双栏布局（侧边栏 + 主内容）
    • 文章列表的多列排版
    • 后台管理系统仪表盘布局

    设计要点

    • 基于 Flexbox 的 12 列栅格，不依赖 float
    • 容器 max-width 980px，符合 Bo-Blog 经典博客宽度
    • 列间距通过 padding 7.5px 实现（左右各半，合计 15px）
    • 无间距模式通过 .boblog-no-gutters 移除所有间距
    • 间距工具类 .boblog-gap-* 使用 CSS gap 属性，适合配合 .boblog-no-gutters 使用
