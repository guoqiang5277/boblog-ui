    12 列网格 (.boblog-row / .boblog-col-*)

    使用说明

    适用场景

    • 博客页面双栏布局（侧边栏 + 主内容）
    • 文章列表的多列排版
    • 后台管理系统仪表盘布局

    设计要点

    • 基于 Flexbox 的 12 列栅格，不依赖 float
    • 容器 max-width 980px，符合 Bo-Blog 经典博客宽度
    • 列间距通过 padding 7.5px 实现（左右各半，合计 15px）
    • 无间距模式通过 .boblog-no-gutters 移除所有间距
    • 间距工具类 .boblog-gap-* 使用 CSS gap 属性，适合配合 .boblog-no-gutters 使用

    宽度容器

    页面内容区居中的宽度限制工具类，自带 margin-left: auto; margin-right: auto 水平居中。

    类名 CSS 效果 使用场景
    .boblog-w-80 width: 80% + 水平居中 FAQ、Help、Terms 页面内容区
    .boblog-max-w-500 max-width: 500px + 水平居中 Login、Register 页面
    .boblog-max-w-600 max-width: 600px + 水平居中 Contact 页面
    .boblog-min-w-280 min-width: 280px 版本切换下拉框最小宽度

    不同宽度容器的居中效果：

    .boblog-w-80（80% 宽度居中）
    .boblog-max-w-500（最大 500px 居中）
    .boblog-max-w-600（最大 600px 居中） 
    <div class="boblog-w-80">80% 宽度居中内容</div>
<div class="boblog-max-w-500">最大 500px 居中内容</div>
<div class="boblog-max-w-600">最大 600px 居中内容</div>
<div class="boblog-min-w-280">最小宽度 280px</div>
    /* 百分比宽度 + 水平居中 — FAQ、Help、Terms 页面内容区 */
.boblog-w-80 { width: 80% !important; margin-left: auto !important; margin-right: auto !important; }

/* 最大宽度 + 水平居中 — Login、Register 页面 */
.boblog-max-w-500 { max-width: 500px !important; margin-left: auto !important; margin-right: auto !important; }

/* 最大宽度 + 水平居中 — Contact 页面 */
.boblog-max-w-600 { max-width: 600px !important; margin-left: auto !important; margin-right: auto !important; }

/* 最小宽度 — 版本切换下拉框等 */
.boblog-min-w-280 { min-width: 280px !important; }
