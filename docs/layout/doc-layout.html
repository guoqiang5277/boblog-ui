    文档布局组件 (Doc Layout)

    概述

    提供两种文档页面布局风格，通过 body class 切换：

    • boblog-doc-gray — 灰底白框，适合概览/规范页面
    • boblog-doc-white — 白底铺开，适合组件详情页

    页面结构固定为三层：

    • <body class="boblog-doc-gray"> — 风格切换
    • <div class="boblog-toc"> — 可选，浮动目录
    • <div class="boblog-doc-container"> — 内容容器

    类名参考

    类名 用途 位置
    boblog-doc-gray 灰底风格 body
    boblog-doc-white 白底风格 body
    boblog-doc-container 内容容器（灰底下为白框，白底下透明） body 直接子级
    boblog-doc-nav 顶部导航链接区域 boblog-doc-container 内
    boblog-doc-section 章节区块 boblog-doc-container 内
    boblog-doc-spec 规范说明块 boblog-doc-section 内
    boblog-doc-demo 演示预览区域 boblog-doc-section 内
    boblog-doc-demo-row 演示行（多行排列时使用） boblog-doc-demo 内

    灰底风格 (boblog-doc-gray)

    • body 背景: #eaeaea（灰色）
    • boblog-doc-container: 白底 + 灰色边框 + 30px 内边距
    • boblog-doc-section: 无背景，50px 底部间距
    • boblog-doc-spec: 浅蓝背景 + 左侧蓝色竖线
    • 适用于: 概览页、规范页（色彩体系、排版规范等）

    白底风格 (boblog-doc-white)

    • body 背景: #FFFFFF（白色），max-width: 1200px，居中
    • boblog-doc-container: 无背景无边框，直接铺开
    • boblog-doc-section: 浅蓝背景面板 + 蓝色边框 + 15px 内边距
    • boblog-doc-spec: 白色背景 + 蓝色边框
    • 适用于: 组件详情页（按钮、表单、表格等）

    风格切换

    两种风格使用完全相同的 HTML 结构，只需切换 body 的 class：

    • <body class="boblog-doc-gray"> → 灰底白框
    • <body class="boblog-doc-white"> → 白底铺开

    内部所有子组件（boblog-doc-container、boblog-doc-section、boblog-doc-spec、boblog-doc-demo）自动适配对应风格，无需修改任何内部结构。

    boblog-doc-gray

    章节标题

    规范说明（浅蓝底 + 左蓝线）
    演示区域

    boblog-doc-white

    章节标题

    规范说明（白底 + 蓝框）
    演示区域

    相同 HTML 结构，仅 body class 不同，子组件自动适配风格。

    <!-- 灰底风格 -->

    与 boblog-toc 配合

    • boblog-toc 放在 boblog-doc-container 外面（body 直接子级）
    • 默认 data-toc-float="true"：浮动覆盖在右上角
    • 设置 data-toc-float="push"：浮动并推开内容
    • boblog-toc 会自动扫描 .boblog-doc-section 内的 h2 标题
