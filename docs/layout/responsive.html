    响应式工具类

    显示/隐藏

    • .boblog-hide-sm: 窗口 < 576px 时隐藏
    • .boblog-hide-md: 窗口 < 768px 时隐藏
    • .boblog-hide-lg: 窗口 < 992px 时隐藏
    • .boblog-hide-xl: 窗口 < 1200px 时隐藏
    • .boblog-show-sm: 窗口 < 576px 时显示（默认隐藏）
    • .boblog-show-md: 窗口 < 768px 时显示（默认隐藏）
    • .boblog-show-lg: 窗口 < 992px 时显示（默认隐藏）
    • .boblog-show-xl: 窗口 < 1200px 时显示（默认隐藏）

    缩放浏览器窗口查看效果（窗口缩小时逐行消失）：

    boblog-hide-xl — 窗口 < 1200px 时隐藏
    boblog-hide-lg — 窗口 < 992px 时隐藏
    boblog-hide-md — 窗口 < 768px 时隐藏
    boblog-hide-sm — 窗口 < 576px 时隐藏

    默认隐藏，窗口小于断点时显示（缩小窗口逐行出现）：

    boblog-show-xl — 窗口 < 1200px 时显示
    boblog-show-lg — 窗口 < 992px 时显示
    boblog-show-md — 窗口 < 768px 时显示
    boblog-show-sm — 窗口 < 576px 时显示

    Flex 工具

    参数说明

    分类 类名 说明
    容器 .boblog-flex display: flex
    .boblog-inline-flex display: inline-flex
    方向 .boblog-flex-row flex-direction: row（默认横向）
    .boblog-flex-row-reverse flex-direction: row-reverse（横向反转）
    .boblog-flex-column flex-direction: column（纵向）
    .boblog-flex-column-reverse flex-direction: column-reverse（纵向反转）
    换行 .boblog-flex-wrap flex-wrap: wrap（允许换行）
    .boblog-flex-nowrap flex-wrap: nowrap（禁止换行）
    主轴对齐 .boblog-justify-start justify-content: flex-start（起点对齐，默认，左对齐）
    .boblog-justify-center justify-content: center（居中对齐）
    .boblog-justify-end justify-content: flex-end（终点对齐）
    .boblog-justify-between justify-content: space-between（两端对齐，项目之间的间隔都相等,左右无边距）
    .boblog-justify-around justify-content: space-around（环绕对齐，每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍,左右边距 = 元素间距的一半）
    .boblog-justify-evenly justify-content: space-evenly（平均分布,元素与容器边缘的间距也相等,左右边距 = 元素间距）
    交叉轴对齐 .boblog-align-start align-items: flex-start（起点对齐，交叉轴的起点对齐）
    .boblog-align-center align-items: center（居中对齐，交叉轴的中点对齐）
    .boblog-align-end align-items: flex-end（终点对齐，交叉轴的终点对齐）
    .boblog-align-stretch align-items: stretch（拉伸填充，默认值，如果项目未设置高度或设为auto，将占满整个容器的高度）
    .boblog-align-baseline align-items: baseline（基线对齐，项目的第一行文字的基线对齐）
    子项 .boblog-flex-1 flex: 1（等分剩余空间）
    .boblog-flex-auto flex: auto（根据内容自动调整）
    .boblog-flex-none flex: none（不伸缩）
    .boblog-flex-grow flex-grow: 1（允许放大）
    .boblog-flex-shrink flex-shrink: 1（允许缩小）
    .boblog-flex-no-grow flex-grow: 0（禁止放大）
    .boblog-flex-no-shrink flex-shrink: 0（禁止缩小）

    演示 - 方向

    Flex 方向决定了主轴的朝向。flex-row 表示主轴为水平方向（项目横向排列），flex-column 表示主轴为垂直方向（项目纵向排列）。带 -reverse 后缀的类会反转排列顺序。

    boblog-flex boblog-flex-row（默认横向）:

    1
    2
    3

    boblog-flex boblog-flex-row-reverse（横向,逆向）:

    1
    2
    3

    boblog-flex boblog-flex-column（纵向）:

    项目 1
    项目 2
    项目 3

    boblog-flex boblog-flex-column-reverse（纵向,逆向）:

    项目 1
    项目 2
    项目 3

    演示 - 交叉轴对齐

    交叉轴是垂直于主轴的方向。当主轴是水平时（flex-row），交叉轴就是垂直方向。交叉轴对齐控制项目在这个垂直方向上的位置（顶部、居中、底部、拉伸等）。

    boblog-flex boblog-align-start（顶部对齐,交叉轴的起点对齐）:

    高30px
    高50px
    高10px
