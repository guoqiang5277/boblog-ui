    前台顶部导航

    基础导航 (.boblog-nav)

    • 容器类名: .boblog-nav
    • 布局: display: flex，align-items: stretch
    • 高度: 由内容撑开（不固定 height），link padding 决定
    • 字体: 13px，Tahoma, Arial, sans-serif
    • 背景色: #FFFFFF (--boblog-bg)
    • 底部粗边框: 10px solid #BEE6FD (--boblog-blue-bar)，与 Bo-Blog V2.1 经典蓝条一致
    • 链接类名: .boblog-nav-link
    • 链接内边距: 5px 15px
    • 链接最小宽度: 72px
    • 链接颜色: #888888 (--boblog-text-muted)，font-weight: bold
    • 链接边框: 1px solid transparent，底部无边框（hover/active 时 margin-bottom: -10px + border-bottom: 10px 融合粗边框）
    • Hover: 文字 #1A1A1A (--boblog-text)，背景 #F4FBFF (--boblog-blue-wash)，边框 #9EC9FF (--boblog-border-blue)，底部 10px solid #E8F0F7
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    下拉子菜单 (.boblog-nav-dropdown)

    • 容器类名: .boblog-nav-dropdown（包裹链接 + 子菜单）
    • 触发方式: 鼠标悬停 (.boblog-nav-dropdown:hover)
    • 下拉三角: 自动在链接右侧追加 ▼ 箭头（::after 伪元素，8px）
    • 焦点保持: hover 子菜单时，标题链接保持激活状态不消失
    • 子菜单类名: .boblog-nav-submenu
    • 子菜单定位: position: absolute，top: calc(100% + 10px)（跳过容器底部粗边框），left: -1px
    • 子菜单最小宽度: 100%（至少与父级 Tab 等宽）
    • 子菜单背景: #FFFFFF (--boblog-bg)
    • 子菜单边框: 1px solid #9EC9FF (--boblog-border-blue)，无顶部边框
    • 子菜单阴影: 2px 2px 5px rgba(0, 0, 0, 0.1)
    • 子菜单层级: z-index: 100 (--boblog-z-index-dropdown)
    • 子项类名: .boblog-nav-submenu-item
    • 子项内边距: 8px 15px
    • 子项文字: #1A1A1A (--boblog-text)，font-weight: normal
    • 子项分隔线: 1px dashed #DDE8F0 (--boblog-border-light)，末项无
    • 子项 Hover: 背景 #F4FBFF (--boblog-blue-wash)，文字 #0D80BD (--boblog-blue-accent)

    嵌套子菜单（二级）

    • 触发元素: .boblog-nav-submenu-item.boblog-nav-has-children（追加类名标记含子菜单）
    • 重要: 含子菜单的项必须使用 <div> 而非 <a>，以避免嵌套 <a> 导致解析异常
    • 右侧箭头: 自动追加 ▶ 箭头（::after 伪元素，8px）
    • 二级菜单定位: position: absolute，top: 0，left: 100%（从右侧弹出）
    • 二级菜单最小宽度: 120px
    • 触发方式: hover .boblog-nav-has-children 时显示

    尺寸变体

    • 小尺寸: .boblog-nav-sm — 字号 12px，底部边框 5px，链接内边距 4px 10px，最小宽度 60px
    • 默认尺寸: .boblog-nav（无额外类名）— 字号 13px，底部边框 10px，链接内边距 5px 15px，最小宽度 72px
    • 大尺寸: .boblog-nav-lg — 字号 14px，底部边框 12px，链接内边距 8px 20px，最小宽度 80px
