    字体定义

    概述

    Bo-Blog UI 引入本地托管的编程等宽字体，用于代码块显示。所有字体均为免费开源（SIL Open Font License）。

    • 格式：woff2（现代浏览器通用，压缩率最高）
    • 子集：Latin（覆盖代码所需的 ASCII + 常用符号）
    • 字体文件位置：dist/fonts/

    预制字体

    项目预置的编程等宽字体（woff2 格式）：

    字体名称 来源 字重 说明
    JetBrains Mono JetBrains 400, 700 辨识度高，主流编程字体
    Fira Code Mozilla 400, 700 连字支持好
    Source Code Pro Adobe 400, 700 清晰稳重
    Cascadia Code Microsoft 400, 700 VS Code 默认字体

    JetBrains Mono

    首选编程字体。来源：GitHub

    @font-face {
    font-family: 'JetBrains Mono';
    font-style: normal;
    font-weight: 400;
    font-display: swap;
    src: url('fonts/JetBrainsMono-Regular.woff2') format('woff2');
}

@font-face {
    font-family: 'JetBrains Mono';
    font-style: normal;
    font-weight: 700;
    font-display: swap;
    src: url('fonts/JetBrainsMono-Bold.woff2') format('woff2');
}

    JetBrains Mono Regular: const foo = (x) => x * 2;
    JetBrains Mono Bold: function bar() { return 42; }

    使用方式

    代码块字体栈已在 CSS 变量中定义，直接使用即可：

    /* 变量定义（typography.css） */
:root {
    --boblog-font-mono: 'JetBrains Mono', 'Fira Code', 'Source Code Pro',
                        'Cascadia Code', Consolas, Monaco, 'Courier New', monospace;
}

/* 使用 */
.boblog-codeblock code {
    font-family: var(--boblog-font-mono);
}
