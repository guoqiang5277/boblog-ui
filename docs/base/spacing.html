    间距/尺寸

    概述

    Bo-Blog UI 使用基于 5px 递增的间距体系，简洁统一。

    • 基础单位：5px
    • 间距级别：6 级（xs, sm, md, lg, xl, 2xl）
    • 边框：1px 标准宽度，无圆角（方角设计）

    间距体系

    基于 5px 递增的间距系统：

    变量名 尺寸 用途 可视化
    --boblog-spacing-xs 5px 极小间距
    --boblog-spacing-sm 10px 小间距
    --boblog-spacing-md 15px 中等间距
    --boblog-spacing-lg 20px 大间距
    --boblog-spacing-xl 25px 特大间距
    --boblog-spacing-2xl 30px 超大间距

    边框

    Bo-Blog 经典风格采用方角设计（无圆角）：

    变量名 值 用途
    --boblog-border-width 1px 标准边框宽度
    --boblog-border-width-thick 2px 粗边框
    --boblog-border-radius 0 无圆角 — 方角设计

    Bo-Blog 特定尺寸

    Bo-Blog 专用组件的固定尺寸：

    变量名 值 用途
    --boblog-date-card-width 49px 日期卡片宽度
    --boblog-date-card-height 46px 日期卡片高度
    --boblog-sidebar-main-width 201px 主侧边栏宽度
    --boblog-sidebar-sub-width 140px 次侧边栏宽度

    使用建议

    • 组件内部小间距：使用 xs (5px) 或 sm (10px)
    • 组件内部标准间距：使用 md (15px)
    • 组件之间间距：使用 lg (20px)
    • 区块之间间距：使用 xl (25px) 或 2xl (30px)
    • 保持视觉层次：不同级别的内容使用不同的间距大小

    间距工具类

    提供快捷的 CSS 类名，无需写内联样式。数字后缀 × 5px。

    命名规则

    前缀 含义 示例
    .boblog-p-* padding 四边 .boblog-p-3 → padding: 15px
    .boblog-pt/pb/pl/pr-* padding 单边（top/bottom/left/right） .boblog-pt-2 → padding-top: 10px
    .boblog-px-* padding 左右 .boblog-px-4 → padding-left/right: 20px
    .boblog-py-* padding 上下 .boblog-py-2 → padding-top/bottom: 10px
    .boblog-m-* margin 四边 .boblog-m-3 → margin: 15px
    .boblog-mt/mb/ml/mr-* margin 单边（top/bottom/left/right） .boblog-mb-4 → margin-bottom: 20px
    .boblog-mx-* margin 左右 .boblog-mx-auto → margin-left/right: auto
    .boblog-my-* margin 上下 .boblog-my-5 → margin-top/bottom: 25px

    数字后缀对照表

    后缀 值 后缀 值 后缀 值
    0 0 5 25px 10 50px
    1 5px 6 30px 11 55px
    2 10px 7 35px 12 60px
    3 15px 8 40px auto auto
    4 20px 9 45px

    Auto 工具类

    类名 效果 典型用途
    .boblog-mx-auto margin-left/right: auto 块元素水平居中
    .boblog-my-auto margin-top/bottom: auto Flex 容器内垂直居中
    .boblog-m-auto margin: auto Flex 容器内完全居中
    .boblog-ml-auto margin-left: auto Flex 子元素推到右侧
    .boblog-mr-auto margin-right: auto Flex 子元素推到左侧
    .boblog-mt-auto margin-top: auto Flex 子元素推到底部
    .boblog-mb-auto margin-bottom: auto Flex 子元素推到顶部
