    排版

    概述

    • 字体族: Tahoma, Arial, sans-serif
    • 基础字号: 12px
    • 基础行高: 1.5 (150%)
    • 代码字体: JetBrains Mono, Fira Code, Source Code Pro, Cascadia Code, Consolas, Monaco, monospace
    • 设计风格: 基于 Bo-Blog V2.1 排版特征，经典的 Tahoma、Arial 字体族

    字号体系

    1. 规范说明

    • 8 级字号体系: 覆盖从极小字到巨大字的全部场景
    • --boblog-font-size-xs: 11px - 极小字
    • --boblog-font-size-sm: 12px - 小字
    • --boblog-font-size-base: 12px - 基础字号（Bo-Blog 标准）
    • --boblog-font-size-md: 14px - 中等字号
    • --boblog-font-size-lg: 16px - 大字
    • --boblog-font-size-xl: 18px - 特大字
    • --boblog-font-size-2xl: 20px - 超大字
    • --boblog-font-size-3xl: 24px - 巨大字

    行高

    • --boblog-line-height-tight: 1.2 - 紧凑行高（标题）
    • --boblog-line-height-normal: 1.5 - 标准行高（Bo-Blog 使用 150%）
    • --boblog-line-height-loose: 1.8 - 宽松行高

    字体族

    • --boblog-font-family: Tahoma, Arial, sans-serif - 正文字体
    • --boblog-font-family-code: 'JetBrains Mono', 'Fira Code', 'Source Code Pro', 'Cascadia Code', Consolas, Monaco, monospace - 代码字体
