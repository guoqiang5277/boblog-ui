    拼音搜索 (Pinyin Search)

    模块说明

    1. 功能说明

    • 支持三种匹配方式：直接文本匹配、拼音首字母匹配、完整拼音匹配
    • 直接文本匹配：搜索词直接包含在目标文本中（如 "技术" 匹配 "技术分享"）
    • 首字母匹配：搜索词为拼音首字母序列（如 "jsfx" 匹配 "技术分享"）
    • 完整拼音匹配：搜索词为完整拼音（如 "jishu" 匹配 "技术"）
    • 导出位置：BoblogUI.pinyin（主命名空间）和 PinyinUtil（向后兼容别名）

    API 说明

    1. matches(text, search)

    判断 text 是否匹配 search，支持直接文本、首字母、完整拼音三种方式。

    参数 类型 说明
    text string 待匹配的目标文本
    search string 搜索词（可为中文、首字母或完整拼音）

    返回值： boolean — 匹配返回 true，否则返回 false

    BoblogUI.pinyin.matches('技术分享', '技术');   // true（直接文本）
BoblogUI.pinyin.matches('技术分享', 'jsfx');   // true（首字母）
BoblogUI.pinyin.matches('技术分享', 'jishu');  // true（完整拼音）
BoblogUI.pinyin.matches('技术分享', 'abc');    // false

    2. getPinyinInitials(str)

    获取字符串中每个汉字的拼音首字母，拼接为首字母序列。

    参数 类型 说明
    str string 输入字符串（可含汉字和非汉字字符）

    返回值： string — 拼音首字母序列（小写）

    BoblogUI.pinyin.getPinyinInitials('技术分享');  // "jsfx"
BoblogUI.pinyin.getPinyinInitials('前端开发');  // "qkdf"
BoblogUI.pinyin.getPinyinInitials('ASP.NET');   // "asp.net"（非汉字原样保留）

    3. getFullPinyin(str)

    获取字符串中每个汉字的完整拼音，拼接为完整拼音序列。

    参数 类型 说明
    str string 输入字符串

    返回值： string — 完整拼音序列（小写，无音调）

    BoblogUI.pinyin.getFullPinyin('技术');  // "jishu"
BoblogUI.pinyin.getFullPinyin('分享');  // "fenxiang"
BoblogUI.pinyin.getFullPinyin('你好');  // "nihao"

    4. getMatchType(text, search)

    返回 text 与 search 的匹配类型，用于精确判断匹配方式。

    参数 类型 说明
    text string 待匹配的目标文本
    search string 搜索词

    返回值： 'direct' | 'initials' | 'fullPinyin' | null

    返回值 含义
    'direct' 直接文本匹配
    'initials' 拼音首字母匹配
    'fullPinyin' 完整拼音匹配
    null 不匹配 
    BoblogUI.pinyin.getMatchType('技术分享', '技术');   // 'direct'
BoblogUI.pinyin.getMatchType('技术分享', 'jsfx');   // 'initials'
BoblogUI.pinyin.getMatchType('技术分享', 'jishu');  // 'fullPinyin'
BoblogUI.pinyin.getMatchType('技术分享', 'xyz');    // null

    5. extend(newMappings)

    扩展拼音映射表，支持添加自定义汉字拼音或覆盖已有映射。

    参数 类型 说明
    newMappings object 汉字到拼音的映射对象，key 为单个汉字，value 为拼音字符串

    返回值： void

    // 添加自定义映射（如生僻字）
BoblogUI.pinyin.extend({
    '㐀': 'qiu',
    '㐁': 'kao'
});

// 覆盖已有映射（如多音字指定读音）
BoblogUI.pinyin.extend({
    '重': 'zhong'   // 默认可能为 'chong'，此处覆盖为 'zhong'
});

    引入方式

    1. 说明

    • 拼音搜索模块已打包进 boblog-ui.js，无需单独引入任何文件
    • 引入 boblog-ui.js 后即可通过 BoblogUI.pinyin 访问所有 API
    • PinyinUtil 为向后兼容的全局别名，与 BoblogUI.pinyin 完全等价

    2. 使用示例

    <!-- 只需引入 boblog-ui.js，无需其他文件 -->
<script src="dist/boblog-ui.js"></script>

<script>
    // 主命名空间（推荐）
    BoblogUI.pinyin.matches('技术分享', 'jsfx');  // true

    // 向后兼容别名
    PinyinUtil.matches('技术分享', 'jsfx');       // true（等价）
</script>
