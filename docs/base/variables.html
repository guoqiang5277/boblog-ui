    CSS 变量

    概述

    Bo-Blog UI 使用 CSS 自定义属性（CSS Variables）统一管理设计令牌（Design Tokens）。

    • 所有变量定义在 :root 选择器中
    • 变量名统一使用 --boblog- 前缀
    • 变量按功能分组：色彩、边框、背景、文字、尺寸等

    基础色

    :root {
    --boblog-white: #FFFFFF;    /* 纯白：主背景、反色文字 */
    --boblog-black: #000000;    /* 纯黑：hover 强调、最深文字 */
}

    使用方式

    /* 在 CSS 中使用变量 */
.my-element {
    color: var(--boblog-text);
    background: var(--boblog-bg);
    border: 1px solid var(--boblog-border);
}

/* 带默认值 */
.my-element {
    color: var(--boblog-text, #333333);
}
