字体体系

规范说明

经典的 Tahoma、Arial 字体族，12px 基础字号，150% 行高。

  • 字体族：Tahoma, Arial, "Microsoft YaHei", sans-serif
  • 基础字号：12px
  • 基础行高：1.5 (150%)

字号展示

xs (10px) - 这是超小字号文本，用于辅助说明
sm (11px) - 这是小号字号文本，用于次要信息
base (12px) - 这是基础字号文本，用于正文内容
md (14px) - 这是中号字号文本，用于小标题
lg (16px) - 这是大号字号文本，用于标题
xl (18px) - 这是超大字号文本，用于主标题
2xl (20px) - 这是特大字号文本，用于页面标题
3xl (24px) - 这是巨大字号文本，用于特殊标题

CSS 变量

/* 字体族 */
--boblog-font-family: Tahoma, Arial, "Microsoft YaHei", sans-serif;

/* 字号 */
--boblog-font-xs: 10px;
--boblog-font-sm: 11px;
--boblog-font-base: 12px;
--boblog-font-md: 14px;
--boblog-font-lg: 16px;
--boblog-font-xl: 18px;
--boblog-font-2xl: 20px;
--boblog-font-3xl: 24px;

/* 行高 */
--boblog-line-height: 1.5;

间距体系

规范说明

统一的间距规范，基于 4px 基础单位。

  • xs: 4px - 最小间距
  • sm: 8px - 小间距
  • md: 16px - 标准间距
  • lg: 24px - 大间距
  • xl: 32px - 超大间距
  • 2xl: 48px - 特大间距

间距展示

xs (4px)
sm (8px)
md (16px)
lg (24px)
xl (32px)
2xl (48px)

CSS 变量

/* 间距 */
--boblog-spacing-xs: 4px;
--boblog-spacing-sm: 8px;
--boblog-spacing-md: 16px;
--boblog-spacing-lg: 24px;
--boblog-spacing-xl: 32px;
--boblog-spacing-2xl: 48px;
