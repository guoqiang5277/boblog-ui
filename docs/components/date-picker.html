    日期选择器 (Date Picker)

    日期选择器 (.boblog-date-picker)

    • 容器: position relative，display inline-block，width 100%
    • 支持类型: input[type="date"]、datetime-local、month、week、time
    • 默认状态: 边框 1px solid #DDE8F0 (--boblog-border-light)，背景 #FFFFFF (--boblog-bg)，内边距 6px 10px
    • Hover: 边框变为 #C4D7E8 (--boblog-border)
    • Focus: 边框 #0D6EAA (--boblog-blue)，背景 #E8F0F7 (--boblog-blue-pale)，outline 2px solid rgba(13, 128, 189, 0.1)
    • 自定义选择器: 引入 boblog-ui.js 后自动转换为自定义日历面板，默认格式 yyyy-MM-dd，可通过 data-format 属性自定义
    • Disabled: 背景 #F5F5F5，边框 #D9D9D9，文字 #888888 (--boblog-text-muted)，cursor not-allowed
    • Readonly: 背景 #F5F8FB (--boblog-bg-subtle)，cursor default
    • Error (.boblog-date-picker-error): 边框 #CC3333 (--boblog-danger)
    • 引入 JS 后原生 input 被隐藏，自动创建显示区域和日历面板

    其他日期类型

    • datetime-local: 日期 + 时间选择，默认格式 yyyy-MM-dd HH:mm
    • month: 年月选择，默认格式 yyyy-MM
    • week: 年周选择，默认格式 YYYY-Www
    • time: 时间选择，默认格式 HH:mm
    • 所有类型共享 .boblog-date-picker 容器样式，JS 自动识别 input 类型并渲染对应面板
    • 点击显示区域弹出自定义选择面板（非浏览器原生 picker），跨浏览器一致

    农历和节气 (data-nongli / data-jieqi)

    • data-nongli="true": 日历格子显示农历日名（初一~三十），第二行灰色 9px 文字
    • data-jieqi="true": 日历格子显示24节气文字行（立春、雨水等），绿色 9px 粗体，格子增高
    • data-jieqi="bg": 节气日以淡绿色背景标记，鼠标悬停显示节气名（title），格子不增高
    • 两个属性独立控制，可单独使用或同时开启
    • 同时开启时（jieqi="true"）：第一行阳历、第二行农历、第三行节气（仅节气日显示第三行）
    • 同时开启时（jieqi="bg"）：第一行阳历、第二行农历，节气日加背景色
    • 影响类型：date、datetime-local、week（含日历网格的面板）
    • 依赖 BoblogUI.lunar 模块（lunar.js），构建时自动包含

    JavaScript API

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描 .boblog-date-picker 中的五种日期/时间 input，转换为自定义选择器
    • 支持的类型：date（日历网格）、datetime-local（日历+时间）、month（月份网格）、week（周选择）、time（时间选择）
    • 各类型默认格式：date → yyyy-MM-dd，datetime-local → yyyy-MM-dd HH:mm，month → yyyy-MM，week → YYYY-Www，time → HH:mm
    • 通过 data-format 属性自定义显示格式，data-placeholder 属性自定义占位文本
    • 原生 input 被隐藏，值由自定义面板同步回 input（保持表单提交功能）
    • 自定义面板完全替代浏览器原生 picker，跨浏览器（Chrome/Firefox/Safari/Edge）一致
    • 支持 disabled、readonly、error 三种状态

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描所有 .boblog-date-picker
    • 无需手动调用任何函数

    2. 手动调用

    // 初始化页面所有日期选择器
BoblogUI.datePicker.init();

// 只初始化指定容器内的日期选择器
BoblogUI.datePicker.init(document.getElementById('myForm'));

    3. 引入方式

    <!-- 引入 CSS（包含日期选择器样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含自定义日历选择器逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 使用 data-format 自定义显示格式 -->
<div class="boblog-date-picker" data-format="yyyy/MM/dd">
    <input type="date" value="2006-08-15">
</div>

<!-- 使用 data-placeholder 自定义占位文本 -->
<div class="boblog-date-picker" data-placeholder="请选择发布日期">
    <input type="date">
</div>

<!-- 默认格式 yyyy-MM-dd -->
<div class="boblog-date-picker">
    <input type="date" value="2006-08-15">
</div>
