    分页组件 (Pagination)

    基础分页 (.boblog-pagination)

    • 容器类名: .boblog-pagination，flex 布局
    • 链接类名: .boblog-page-link
    • 文字颜色: #1A1A1A (--boblog-text)
    • 背景: #FFFFFF (--boblog-bg)
    • 边框: 1px solid #9EC9FF (--boblog-border-blue)
    • 内边距: 4px 10px
    • 最小宽度: 32px，文本居中
    • 悬停: 背景 #F4FBFF (--boblog-blue-wash)，文字 #0D80BD (--boblog-blue-accent)，边框 #0D80BD
    • 间距: 页码间隔 5px (--boblog-spacing-xs)
    • 字号: 12px (--boblog-font-size-sm)
    • 本组件为纯 CSS 实现，无需引入 JavaScript
