    树形组件

    基础用法

    • 容器类名: .boblog-tree
    • 节点: .boblog-tree-node（加 .expanded 展开）
    • 节点内容: .boblog-tree-node-content（flex 布局）
    • 展开按钮: .boblog-tree-toggle（▶ 图标，展开时旋转 90deg 变 ▼）
    • 叶子节点: .boblog-tree-toggle.leaf（隐藏图标但保留占位）
    • 节点文字: .boblog-tree-label
    • 子节点容器: .boblog-tree-children（默认隐藏，展开时显示）
    • 展开/折叠: 通过 JavaScript 切换 .boblog-tree-node 的 .expanded 类（详见 JavaScript API）

    选中与禁用

    • 选中状态: .selected（加在 .boblog-tree-node-content 上）
    • 选中效果: 浅蓝背景 + 蓝色加粗文字
    • 禁用状态: .disabled（加在 .boblog-tree-node 上）
    • 禁用效果: 半透明 + 不可点击
    • 交互: 点击节点内容时切换 .selected 类
    分类
    技术文章

    链接标签

    • 用法: 将 .boblog-tree-label 从 <span> 换成 <a> 标签
    • 默认样式: 去掉下划线，继承深色文字（与普通 span 标签一致）
    • hover 状态: 蓝色文字 + 下划线
    • 激活状态: 加 .active 类，蓝色文字 + 加粗
    • 适用场景: 分类目录树、导航目录等需要点击跳转的节点
    文章分类
    技术博客
    生活随笔
    读书笔记 
    <!-- 链接标签：将 span 换成 a 标签 -->
<div class="boblog-tree-node">
    <div class="boblog-tree-node-content">
        <span class="boblog-tree-toggle leaf"></span>
        <a href="/category/tech" class="boblog-tree-label">技术博客</a>
    </div>
</div>

<!-- 激活状态：加 .active 类 -->

    1. 自动初始化

    • 引入 boblog-ui.js 后，点击 .boblog-tree-node-content 自动展开/折叠子节点
    • 基于事件委托实现，无需手动调用任何函数
    • 叶子节点（无 .boblog-tree-children）点击不做展开/折叠处理

    2. 引入方式

    <!-- 引入 CSS -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含 tree.js 展开/折叠交互） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/tree.js"></script>
    • boblog-ui.js 已包含 tree.js，无需额外引入
    • tree.js 使用事件委托监听 document 上的 click 事件，对有子节点的 .boblog-tree-node 切换 .expanded 类
