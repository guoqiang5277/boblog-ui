    手动斑马纹 - 合并单元格 (.boblog-table-striped-manual)

    • 附加类名: .boblog-table-striped-manual（与 .boblog-table 配合使用）
    • 适用场景: 表格中有 rowspan 合并单元格时，普通斑马纹会错乱，使用此类可自动计算正确的斑马纹
    • 工作原理: JS 自动扫描表格结构，根据第一列的 rowspan 计算视觉行号，自动添加偶数行背景色
    • 依赖: 需要引入 boblog-ui.js

    排序 + 分页 (data-page-size)

    • 排序: .boblog-th-sortable 加在 th 上，点击切换 无序 → 升序 → 降序 → 无序
    • 分页: data-page-size="N" 加在 table 上，自动生成分页导航
    • 联动: 排序后自动跳回第 1 页
    • 数据类型: 自动识别数字、日期(YYYY-MM-DD)、字符串
    • 性能上限: 纯前端方案，建议数据量 ≤ 5000 行（1000 行内流畅，5000 行排序有轻微延迟；万行以上 DOM 节点过多导致页面卡顿，不建议使用；十万/百万级数据需服务端分页）
    • 需引入 boblog-ui.js（详见 JavaScript API）

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描：
    • 排序：所有 .boblog-th-sortable 表头，绑定点击排序事件
    • 分页：所有带 data-page-size 的表格，自动生成分页导航
    • 排序后自动跳回第 1 页，排序与分页联动

    2. 手动调用

    // 初始化页面所有可排序/分页表格
BoblogUI.table.init();

// 只初始化指定容器内的表格
BoblogUI.table.init(document.getElementById('myContainer'));

    3. 引入方式

    <!-- 引入 CSS（包含表格 + 分页样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含排序 + 分页逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/table.js"></script>
