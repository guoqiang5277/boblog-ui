    引用 (Blockquote)

    本组件为纯 CSS 实现，无需引入 JavaScript

    基础用法

    规格说明

    • 使用 .boblog-blockquote 类创建基础引用块
    • 使用 .boblog-blockquote-footer 添加引用来源
    • 左侧蓝色竖线边框，浅蓝色背景
    • 适当的内边距和间距
    学而不思则罔，思而不学则殆。
    孔子《论语》
    <blockquote class="boblog-blockquote">
    学而不思则罔，思而不学则殆。
    <div class="boblog-blockquote-footer">孔子《论语》</div>
</blockquote>
    <!DOCTYPE html>
<html lang="zh-CN">
<head>

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>示例</title>
    <link rel="stylesheet" href="../boblog-ui.css">
</head>
<body>
    <blockquote class="boblog-blockquote">
        学而不思则罔，思而不学则殆。
        <div class="boblog-blockquote-footer">孔子《论语》</div>
    </blockquote>
</body>
</html>

    大号引用

    规格说明

    • 添加 .large 类创建大号引用
    • 字号更大，内边距更宽松
    • 适合重要内容或长文引用
    人生天地之间，若白驹过隙，忽然而已。故圣人不贵尺之璧，而重寸之阴，时难得而易失也。
    《淮南子·原道训》

    居中引用

    规格说明

    • 添加 .center 类创建居中引用
    • 文字居中对齐
    • 移除左侧边框，添加上下边框
    • 适合作为独立的引用段落
    天行健，君子以自强不息；地势坤，君子以厚德载物。
    《周易》
    <blockquote class="boblog-blockquote center">
    天行健，君子以自强不息；地势坤，君子以厚德载物。

    带引号图标

    规格说明

    • 添加 .with-icon 类显示引号图标
    • 在内容前显示大号引号符号
    • 增加视觉装饰效果
    • 可与其他修饰类组合使用
    知之为知之，不知为不知，是知也。
    孔子《论语》
    <blockquote class="boblog-blockquote with-icon">
