    模态框

    基础用法

    • 遮罩层: .boblog-modal-backdrop（全屏固定，半透明黑底，弹性居中）
    • 模态框容器: .boblog-modal（白底蓝边，默认自适应：宽 640px，高度由内容撑开）
    • 三段式结构: .boblog-modal-header / .boblog-modal-body / .boblog-modal-footer
    • 头部标题: .boblog-modal-title（默认 14px 加粗）
    • 关闭按钮: .boblog-modal-close（默认 20×20，右侧 × 号，hover 变深）
    • 头/底背景: #F5F8FB (--boblog-bg-subtle)
    • 边框: #9EC9FF (--boblog-border-blue)
    • 分隔线: #DDE8F0 (--boblog-border-light)

    尺寸变体

    尺寸 类名 宽度 固定高度 头部高 底部高 标题字号 内容字号 适用场景
    小 .boblog-modal-sm 240px 100px 28px 28px 12px 12px 简单提示、确认操作
    默认 .boblog-modal 640px 360px 42px 50px 14px 13px 一般对话框
    大 .boblog-modal-lg 1080px 720px 56px 68px 18px 15px 复杂表单、表格
    超大 .boblog-modal-xl 1440px 960px 72px 88px 22px 17px 数据展示、编辑器

    尺寸类加在 .boblog-modal 元素上。默认自适应（高度由内容撑开），尺寸类仅控制宽度和子元素样式。

    叠加 .boblog-modal-fixed 可启用固定模式：宽高均固定。例如 .boblog-modal.boblog-modal-lg.boblog-modal-fixed。

    提示

    sm 自适应，宽 240px，高度由内容撑开。

    默认尺寸

    默认自适应，宽 640px，高度由内容撑开。

    详细信息

    lg 自适应，宽 1080px，高度由内容撑开。

    数据展示

    xl 自适应，宽 1440px，高度由内容撑开。

    sm + fixed

    小尺寸固定，240×100。

    默认 + fixed

    默认尺寸固定，640×360。

    lg + fixed

    大尺寸固定，1080×720。

    xl + fixed

    超大尺寸固定，1440×960。

    固定模式

    • 类名: .boblog-modal-fixed（加在 .boblog-modal 上）
    • 效果: 宽高均固定（默认 640×360），取消视口限制
    • 可叠加: 配合尺寸类（sm/lg/xl）使用，各有对应固定宽高
    • 对比: 默认行为是自适应（高度由内容撑开），加 fixed 后变为固定

    固定模态框

    固定尺寸 640×360，高度不随内容变化。

    固定模态框（lg）

    固定尺寸 1080×720，高度不随内容变化。

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描页面中的模态框
    • 自动绑定 .boblog-modal-close 关闭按钮的点击事件
    • 自动绑定遮罩层空白区域（.boblog-modal-backdrop）的点击关闭事件
    • 自动绑定 ESC 键关闭当前打开的模态框
    • 打开时自动禁止 body 滚动，关闭后恢复

    2. 手动调用

    // 显示模态框（参数为 backdrop 元素的 ID）
BoblogUI.Modal.show('myModal');

// 隐藏模态框
BoblogUI.Modal.hide('myModal');

// 切换显示/隐藏
BoblogUI.Modal.toggle('myModal');

// 手动初始化新添加的模态框（动态 DOM 场景）
BoblogUI.Modal.init(document.getElementById('container'));
    方法 参数 说明
    show(id) backdrop 元素的 ID 显示模态框，禁止 body 滚动
    hide(id) backdrop 元素的 ID 隐藏模态框，无其他模态框时恢复 body 滚动
    toggle(id) backdrop 元素的 ID 切换显示/隐藏状态
    init([container]) HTMLElement（可选，默认 document） 初始化指定容器内模态框的事件绑定

    3. 引入方式

    <!-- CSS -->
<link rel="stylesheet" href="boblog-ui.css">
<!-- JS（必须引入，提供 Modal show/hide/toggle API 和自动事件绑定） -->
<script src="boblog-ui.js"></script>
