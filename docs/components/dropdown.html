    下拉菜单组件

    基础用法

    • 容器: .boblog-dropdown（相对定位，inline-block）
    • 触发按钮: .boblog-dropdown-toggle（点击切换面板显示/隐藏）
    • 下拉面板: .boblog-dropdown-menu（默认隐藏，加 .show 类后显示）
    • 菜单项: .boblog-dropdown-item（块级，hover 时浅蓝背景）
    • 分割线: .boblog-dropdown-divider（水平分割线）
    • 默认等宽: 按钮与面板等宽，由两者中较宽者决定容器宽度
    • 对齐方式: 面板默认右对齐（right: 0），符合顶栏右侧菜单布局

    自适应模式

    • 默认等宽: 不加类名时，按钮与面板等宽（由 JS 自动计算面板宽度并锁定容器 min-width）
    • 自适应模式: 在 .boblog-dropdown 上加 .boblog-dropdown-auto，按钮与面板各自独立宽度
    • 适用场景: 按钮文字较短、面板内容较多，不需要强制等宽时使用

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，Dropdown 会在 DOMContentLoaded 时自动初始化等宽模式
    • JS 会扫描所有 .boblog-dropdown（排除 .boblog-dropdown-auto），测量面板宽度并设为容器 min-width
    • 展开/关闭交互仍需业务方自行绑定 toggle 点击事件，切换 .boblog-dropdown-menu 的 .show 类

    2. 手动调用

    // 重新初始化等宽模式（动态插入新 Dropdown 后调用）
BoblogUI.dropdown.init();

// 仅初始化指定容器内的 Dropdown
BoblogUI.dropdown.init(document.querySelector('#my-container'));

// 打开下拉菜单
var menu = document.querySelector('#my-dropdown .boblog-dropdown-menu');
menu.classList.add('show');

// 关闭下拉菜单
menu.classList.remove('show');

// 切换下拉菜单
menu.classList.toggle('show');

// 点击页面空白处关闭所有菜单
document.addEventListener('click', function(e) {
    if (!e.target.closest('.boblog-dropdown')) {
        document.querySelectorAll('.boblog-dropdown-menu.show').forEach(function(m) {
            m.classList.remove('show');
        });
    }
});

    使用说明

    适用场景：

    • 后台顶栏用户菜单（头像右侧的操作菜单）
    • 前台导航栏帮助菜单、用户菜单
    • 任何需要点击弹出操作列表的场景

    设计要点：

    • Bo-Blog 经典风格：无圆角，Tahoma 字体，12px 基础字号
    • 默认等宽模式：按钮与面板等宽，由两者中较宽者决定容器宽度
    • 自适应模式（.boblog-dropdown-auto）：按钮与面板各自独立宽度
    • 默认右对齐，符合顶栏右侧菜单的常见布局习惯
    • toggle hover 使用半透明白色叠加（rgba(255,255,255,0.12)），不限定底色，深浅背景均可用
    • 移动端保持浮层弹出，菜单项 padding 增大以优化触摸体验

    CSS 变量：

    • --boblog-bg-white — 下拉面板背景色
    • --boblog-border-light — 面板边框颜色
    • --boblog-shadow-light — 面板投影
    • --boblog-z-index-dropdown — 面板层叠顺序
    • --boblog-font-size-base — 基础字号（12px）
    • --boblog-font-family — 字体栈（Tahoma, Arial）
    • --boblog-spacing-sm — 菜单项内边距
    • --boblog-text-dark — 菜单项文字颜色
    • --boblog-bg-content — 移动端面板背景色
    • --boblog-blue-accent — 菜单项 hover 背景色
    • --boblog-text-gray — 禁用菜单项文字颜色
    • --boblog-transition-normal — 过渡时长
    • --boblog-transition-timing — 过渡缓动函数
    • --boblog-line-height-normal — 行高

    注意事项：

    • 点击页面其他区域应关闭已打开的菜单，可通过监听 document click 事件实现
    • 同一页面多个下拉菜单互斥时，打开一个应关闭其他，需业务层自行处理
    • 默认等宽模式由 JS 自动初始化（测量面板宽度 → 设为容器 min-width），按钮 width:100% 撑满容器
    • 自适应模式（.boblog-dropdown-auto）跳过 JS 初始化，按钮和面板各自独立宽度
    • 面板使用 absolute 定位浮于内容之上，父容器需确保有足够的 overflow 可见区域
