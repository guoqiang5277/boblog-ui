    时间轴 (Timeline)

    基础用法

    • 容器类名: .boblog-timeline
    • 列表样式: list-style: none，去除默认列表符号
    • 列表项: .boblog-timeline-item，position: relative，padding-left: 24px，padding-bottom: 20px
    • 垂直线: .boblog-timeline-item::before 伪元素，绝对定位，左侧 5px，宽度 1px，高度 100%，背景色 #D3E7FC (--boblog-border-light)
    • 时间点: .boblog-timeline-dot，11px × 11px 方形，边框 2px solid #0D6EAA (--boblog-blue)，背景 #FFFFFF (--boblog-bg)，绝对定位 left: 0，top: 2px
    • 内容: .boblog-timeline-content，字体大小 12px，颜色 #333333 (--boblog-text)
    • 时间标签: .boblog-timeline-time，字体大小 11px，颜色 #888888 (--boblog-text-secondary)
    • 发布了 1.0 版本

    • 新增评论功能

    颜色变体

    • 修饰类: .success / .warning / .danger / .info（添加到 .boblog-timeline-item）
    • success: 圆点边框色 #52C41A (--boblog-success)，垂直线色 #B7EB8F
    • warning: 圆点边框色 #FAAD14 (--boblog-warning)，垂直线色 #FFE58F
    • danger: 圆点边框色 #F5222D (--boblog-danger)，垂直线色 #FFA39E
    • info: 圆点边框色 #1890FF (--boblog-info)，垂直线色 #91D5FF
    • 适用场景: 不同状态或优先级的事件标记
    • 部署成功
      2024-05-10 10:30
    • 发现性能警告
      2024-05-12 14:20
    • 系统错误，紧急修复
      2024-05-15 09:15
    • 用户数突破 10000
      2024-05-20 16:00

    紧凑模式

    • 修饰类: .boblog-timeline-compact（添加到 .boblog-timeline）

    • 适用场景: 需要在有限空间内展示更多历史记录
