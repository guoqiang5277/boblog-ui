Tab 切换面板 (Tabs)

目录

    基础用法

    1. 规范说明

    • 容器类名: .boblog-tabs
    • 按钮栏: .boblog-tabs-bar（水平排列）
    • 按钮: .boblog-tabs-btn（加 .active 为激活状态）
    • 面板: .boblog-tabs-panel（加 .active 为显示状态）
    • 按钮与面板按顺序一一对应（第1个按钮 → 第1个面板）
    • 按钮默认: 浅蓝背景 var(--boblog-blue-wash) + 蓝框 var(--boblog-border-blue)
    • 按钮栏底线: 1px solid var(--boblog-border-blue)
    • 激活按钮: 白底 + 白色底边遮盖底线（与面板连通）+ 蓝色文字 var(--boblog-blue-accent)
    • 面板: 白底 + 蓝框 var(--boblog-border-blue)，顶部无边框
    • 自动功能: DOMContentLoaded 自动绑定点击切换

    自定义宽度

    1. 规范说明

    • .boblog-tabs 默认宽度 100%（撑满父容器）

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描 .boblog-tabs
    • 无需手动调用任何函数
    • 已初始化的容器不会重复绑定（通过 data-boblog-tabs-init 标记）

    2. 手动调用

    // 初始化页面所有 Tab
BoblogUI.tabs.init();

// 只初始化指定容器内的 Tab
BoblogUI.tabs.init(document.getElementById('myContainer'));

    3. 引入方式

    <!-- 引入 CSS（包含 Tab 样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含 Tab 逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/tabs.js"></script>
