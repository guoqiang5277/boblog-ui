    面板

    基础用法

    • 容器: .boblog-panel（蓝色边框，白底，底部 margin 14px）
    • 三段式结构: .boblog-panel-header / .boblog-panel-body / .boblog-panel-footer
    • 头部标题: .boblog-panel-title（14px 加粗）
    • 头/底背景: --boblog-bg-panel（浅蓝）
    • 边框: --boblog-blue-border
    • 分隔线: --boblog-blue-border-list

    面板标题

    这是面板的主体内容区域。

    可以包含任意内容，如文字、表单、图片等。

    <div class="boblog-panel">
    <div class="boblog-panel-header">
        <h3 class="boblog-panel-title">面板标题</h3>
    </div>
    <div class="boblog-panel-body">
        <p>这是面板的主体内容区域。</p>
        <p>可以包含任意内容，如文字、表单、图片等。</p>
    </div>
    <div class="boblog-panel-footer">
        <button class="boblog-btn">取消</button>
        <button class="boblog-btn boblog-btn-primary">确定</button>
    </div>
</div>

    仅标题和内容

    • 用途: 省略底部区域，纯信息展示场景
    • 结构: 只包含 .boblog-panel-header + .boblog-panel-body

    嵌套表单

    • 用途: 面板内嵌套表单等复杂内容
    • 结构: .boblog-panel-body 内放置 .boblog-form-group

    用户信息

    <div class="boblog-panel">
    <div class="boblog-panel-header">
        <h3 class="boblog-panel-title">用户信息</h3>
    </div>
    <form class="boblog-form boblog-form-horizontal">
        <div class="boblog-panel-body">
            <div class="boblog-form-group">
                <label class="boblog-form-label">用户名</label>
                <div class="boblog-form-control-wrap">
                    <input type="text" class="boblog-input" placeholder="请输入用户名">
                </div>
            </div>
            <div class="boblog-form-group">
                <label class="boblog-form-label">邮箱</label>
                <div class="boblog-form-control-wrap">
                    <input type="email" class="boblog-input" placeholder="请输入邮箱">
                </div>
            </div>
        </div>
        <div class="boblog-panel-footer boblog-panel-center">
            <button type="reset" class="boblog-btn">重置</button>
            <button type="submit" class="boblog-btn boblog-btn-primary">保存</button>
        </div>
    </form>
</div>
