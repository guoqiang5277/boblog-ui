    徽章 (Badge)

    基础用法

    • 基础类名: .boblog-badge
    • 内边距: 2px 8px
    • 字体大小: 11px，Tahoma, Arial, sans-serif
    • 默认样式: 边框 1px solid #9EC9FF (--boblog-border-blue)，背景 #FFFFFF (--boblog-bg)，文字 #0D6EAA (--boblog-blue)
    • 颜色修饰类: .primary / .success / .warning / .danger / .info / .gray
    • 颜色变体为实底: 对应色背景 + 白色文字

    圆点徽章

    • 修饰类: .dot
    • 实现方式: ::before 伪元素生成 8px 彩色指示器
    • 默认形状: 方形（与 Bo-Blog 无圆角设计一致）
    • 圆形指示器: 追加 .round 修饰类，圆点变为圆形
    • 圆点颜色: 继承 currentColor（随文字颜色变化）
    • 垂直对齐: flex 布局居中对齐
    • 适用场景: 在线/离线等状态指示

    方形圆点（默认）：

    在线 离线 忙碌 空闲

    圆形圆点（加 .round）：

    计数器徽章

    • 容器类名: .boblog-badge-counter（包裹目标元素 + 角标）
    • 角标定位: position: absolute，top: 0，left: 100%，transform: translateY(-50%)
    • 角标尺寸: 最小宽 14px，高 14px，字号 9px，内边距 0 4px
    • 适用场景: 通知数、未读消息等角标
