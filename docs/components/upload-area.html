    上传区域 (Upload Area)

    基础用法

    虚线边框 + 浅蓝背景，支持点击选择文件和拖拽上传。鼠标悬停时背景变深，拖拽文件进入时进一步加深背景。

    • 结构: .boblog-upload-area > .boblog-icon-upload + .boblog-text-md + .boblog-text-xs.boblog-text-gray
    • 边框: 2px dashed (--boblog-blue-accent)
    • 背景: --boblog-bg-content (#F4FBFF)
    • hover 背景: --boblog-blue-pale (#E8F0F7)
    • dragging 背景: --boblog-border-blue-lighter (#D3E7FC)
    • 关联输入框: 通过 data-file-input 属性指定选择器
    📁
    点击或拖拽文件到此处上传
    支持图片、音频、视频、附件等多种格式
    <div class="boblog-upload-area" data-file-input="#demoFileInput">
    <div class="boblog-icon-upload">📁</div>
    <div class="boblog-text-md">点击或拖拽文件到此处上传</div>
    <div class="boblog-text-xs boblog-text-gray">支持图片、音频、视频、附件等多种格式</div>
</div>
<input type="file" id="demoFileInput" multiple hidden />

    JavaScript API

    1. 自动初始化

    页面加载后，boblog-ui.js 自动扫描所有 .boblog-upload-area 元素，绑定拖拽和点击事件。上传区域通过 data-file-input 属性指定关联的文件输入框选择器。

    <!-- data-file-input 指向关联的 input[type="file"] -->
<div class="boblog-upload-area" data-file-input="#myFileInput">
    <div class="boblog-icon-upload">📁</div>
    <div class="boblog-text-md">点击或拖拽文件到此处上传</div>
    <div class="boblog-text-xs boblog-text-gray">支持多种格式</div>
</div>
<input type="file" id="myFileInput" multiple hidden />

    2. 手动调用

    如需在动态生成的元素上初始化，或需要监听文件选择回调，可手动调用 API。

    // 初始化上传区域
BoblogUI.uploadArea.init({
    uploadAreaEl: '#uploadArea',
    fileInputEl: '#fileInput',
    onFilesSelected: function(files) {
        // 处理选中的文件
    }
});

// 工具函数
BoblogUI.uploadArea.formatFileSize(1024);        // "1 KB"
BoblogUI.uploadArea.getFileIcon('image/png');     // "🖼️"
BoblogUI.uploadArea.getFileIconByType('video');   // "🎬"

// 消息提示
var msgEl = document.getElementById('message');
BoblogUI.uploadArea.showMessage(msgEl, '上传成功', 'success');

// 列表容器显隐
var listEl = document.getElementById('fileList');
BoblogUI.uploadArea.updateListVisibility(listEl, true);
