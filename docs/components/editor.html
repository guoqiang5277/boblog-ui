    富文本编辑器 (Editor)

    快速开始

    <!-- CSS -->
<link rel="stylesheet" href="../boblog-ui.css">

<!-- JS: 先引入 TinyMCE，再引入 boblog-ui -->
<script src="/lib/tinymce/tinymce.min.js"></script>
<script src="../boblog-ui.js"></script>
    <div class="boblog-editor" data-toolbar="simple">
    <textarea class="boblog-editor-content"><p>初始内容</p></textarea>
</div>

    页面加载后自动初始化，无需手动调用。

    var editorEl = document.querySelector('.boblog-editor');
var html = BoblogUI.editor.getContent(editorEl);

    配置项

    通过 data 属性配置编辑器：

    属性 说明 默认值
    data-toolbar 工具栏预设（basic / simple / standard / full） full
    data-height 编辑器高度（像素） 500
    data-menubar 是否显示菜单栏（"false" 隐藏） true
    data-upload-url 图片上传接口 URL 无（禁用上传）
    data-base-url TinyMCE 资源路径 自动推断
    data-language 语言代码（zh_CN / zh_TW / en） zh_CN

    配置示例

    <div class="boblog-editor"
     data-toolbar="simple"
     data-height="300"
     data-menubar="false"
     data-upload-url="/api/upload/image">
    <textarea class="boblog-editor-content"></textarea>
</div>

    工具栏预设

    预设值 说明
    basic 基础格式：撤销重做、加粗斜体、列表、链接
    simple 推荐 - 简洁款：格式、标题、列表、引用、链接、代码、颜色、字体、字号、行距、清除格式
    standard 标准格式：含对齐、图片、清除格式
    full 完整功能：4行工具栏，含所有功能（上传、表格、表情等）

    simple 工具栏（推荐）

    上传方式

    • 工具栏按钮: 在 data-toolbar 中添加 uploader（full 预设已包含）
    • 粘贴上传: 直接粘贴剪贴板中的图片
    • 拖拽上传: 将图片拖入编辑区域

    服务端接口要求

    请求方式 POST
    Content-Type multipart/form-data
    表单字段 file（文件）、fileType（固定值 "Images"）

    返回 JSON 格式：

    // 成功
{ "location": "/uploads/2024/01/image.jpg" }

// 失败
{ "error": "文件大小超出限制" }

    JavaScript API

    方法 说明
    BoblogUI.editor.init(container) 初始化容器内的编辑器（用于动态插入的内容）
    BoblogUI.editor.destroy(editorEl) 销毁编辑器实例
    BoblogUI.editor.getContent(editorEl) 获取编辑器内容（返回 HTML 字符串）
    BoblogUI.editor.setContent(editorEl, html) 设置编辑器内容

    获取内容并提交

    // 获取编辑器容器
var editorEl = document.querySelector('.boblog-editor');

// 获取 HTML 内容
var content = BoblogUI.editor.getContent(editorEl);

// 提交到服务器
fetch('/api/article/save', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ content: content })
});

    表单提交

    编辑器在 form 表单内时，TinyMCE 会自动同步内容到 textarea：

    <form action="/api/article/save" method="post">
    <div class="boblog-editor">
        <textarea name="content" class="boblog-editor-content"></textarea>
    </div>
    <button type="submit">保存</button>
</form>
