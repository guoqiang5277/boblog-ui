    树形选择器 (Tree Select)

    基础用法

    • 最外层容器: .boblog-tree-select（inline-block，相对定位，作为下拉面板的定位锚点）
    • 显示框: .boblog-tree-select-display + .boblog-input（flex 布局，左侧显示选中名称，右侧显示箭头）
    • 选中名称元素: 需添加 data-display-name 属性，JS 通过此属性更新显示文字
    • 下拉箭头: .boblog-tree-select-arrow（▼）
    • 下拉面板: .boblog-tree-select-dropdown（绝对定位，默认隐藏，展开时加 .show）
    • 搜索框区域: .boblog-tree-select-search（面板顶部固定，下方有分隔线）
    • 树节点容器: .boblog-tree-select-list
    • 单个树节点: .boblog-tree-select-item（flex 布局，可追加 .selected / .directory-type）
    • 展开/折叠按钮: .tree-item-toggle（无子节点时加 .invisible，折叠时加 .collapsed）
    • 节点内容区: .tree-item-content > .tree-item-header（flex 布局，包含名称、别名、计数）
    • 节点名称: .boblog-tree-select-name
    • 别名标签: .boblog-tree-select-slug（浅蓝底灰字）
    • 计数徽章: .boblog-tree-select-count（有文章时加 .has-articles，蓝底白字）
    • 节点描述: .tree-item-desc（灰色小字，可选）
    • 类型徽章: .tree-item-type-badge（可追加 .type-directory / .type-final）
    • 搜索高亮: .highlight（黄底，由 JS 自动注入）
    • 无匹配提示: .no-results-message（由 JS 自动注入）
    • 层级缩进: 由 JS 通过 padding-left 动态计算（每层 16px）
    • 初始化: 需调用 BoblogTreeSelect.create(config) 完成功能绑定（见 API 说明章节）

    API 说明

    • 入口方法: BoblogTreeSelect.create(config) — 创建并初始化一个树形选择器实例
    • 返回值: 实例对象，可调用公共方法

    create() 配置项

    参数 类型 必填 说明
    container HTMLElement 是 最外层 .boblog-tree-select 容器 DOM 元素
    treeData Array 是 树形数据数组，每项结构见下方"节点数据格式"
    selectedId number | null 否 初始选中的节点 ID，null 表示未选中，默认 null
    onSelect Function 否 选中回调，签名：function(id, name, fullPath)。id 为节点 ID（number），name 为节点名称，fullPath 为完整路径字符串（如"前端开发 > CSS"）
    placeholder string 否 未选中时显示框的占位文字，默认 '请选择'
    emptyOptionText string | null 否 空选项文字（如"-- 不限分类 --"），null 表示不显示空选项，默认 null
    nodeMatches Function | null 否 自定义搜索匹配函数，签名：function(flatNode, search) → boolean。不传时默认使用拼音匹配（支持首字母/完整拼音/直接文本），同时匹配 name 和 slug
    renderNodeExtra Function | null 否 自定义节点额外 HTML 渲染函数，签名：function(flatNode) → string。返回的 HTML 会插入节点名称之后，可用于追加类型徽章等
    isSelectable Function | null 否 判断节点是否可选函数，签名：function(flatNode) → boolean。返回 false 时节点不响应点击选中，并添加 .directory-type 样式（弱化显示）。默认：有子节点的为目录不可选，叶子节点可选

    节点数据格式（treeData 每项）

    字段 类型 必填 说明
    id number 是 节点唯一 ID
    name string 是 节点显示名称
    slug string 否 别名，显示为浅蓝底灰字标签，传空字符串或不传则不显示
    description string 否 节点描述，显示为节点名称下方的灰色小字，不传则不显示
    type string 否 节点类型，传入值存入 flatNode.type，可配合 renderNodeExtra 渲染类型徽章，默认 'final'
    articleCount number 否 文章数量，大于 0 时计数徽章显示蓝底白字（.has-articles），默认 0
    children Array 否 子节点数组，结构与父节点相同，支持多级嵌套，不传或空数组则为叶子节点

    实例公共方法

    方法 返回值 说明
    getSelectedId() number | null 获取当前选中节点的 ID，未选中时返回 null
    selectById(id) void 程序化选中指定 ID 的节点，同时触发 onSelect 回调并更新显示框文字
    refreshData(treeData) void 刷新树形数据并重新渲染节点列表，用于动态更新数据场景 
    // 创建实例并保存引用
var treeSelect = BoblogTreeSelect.create({
    container: document.getElementById('myTreeSelect'),
    treeData: [
        {
            id: 1, name: '技术文章', slug: 'tech', type: 'directory',
            articleCount: 0,
            description: '目录型分类，不可直接归档',
            children: [
                { id: 2, name: 'JavaScript', slug: 'js', type: 'final', articleCount: 12 },
                { id: 3, name: 'C#',         slug: 'csharp', type: 'final', articleCount: 8 }
            ]
        },
        { id: 4, name: '生活随笔', slug: 'life', type: 'final', articleCount: 5 }
    ],
    selectedId: 2,                    // 初始选中 JavaScript
    placeholder: '请选择分类',
    emptyOptionText: '不限分类',       // 显示空选项
    // 自定义：目录型分类不可选
    isSelectable: function(node) {
        return node.type !== 'directory';
    },
    // 自定义：追加类型徽章 HTML
    renderNodeExtra: function(node) {
        if (node.type === 'directory') {
            return '<span class="tree-item-type-badge type-directory">目录</span>';
        }
        if (node.type === 'final') {
            return '<span class="tree-item-type-badge type-final">归档</span>';
        }
        return '';
    },
    // 自定义：搜索时同时匹配名称和别名
    nodeMatches: function(node, search) {
        var s = search.toLowerCase();
        return node.name.toLowerCase().indexOf(s) !== -1
            || (node.slug && node.slug.toLowerCase().indexOf(s) !== -1);
    },
    onSelect: function(id, name, path) {
        console.log('选中节点 ID:', id);
        console.log('节点名称:', name);
        console.log('完整路径:', path);  // 如 "技术文章 > JavaScript"
        document.getElementById('hiddenCategoryId').value = id || '';
    }
});

// 程序化选中节点
treeSelect.selectById(3);

// 获取当前选中 ID
var currentId = treeSelect.getSelectedId();

// 动态刷新数据
treeSelect.refreshData([
    { id: 10, name: '新分类', slug: 'new', articleCount: 0 }
]);
