手风琴折叠面板 (Accordion)

目录

    参数说明

    1. CSS 类名

    类名 用在 说明
    .boblog-accordion 容器 手风琴最外层容器，蓝色边框
    .boblog-accordion-item 折叠项 单个折叠项，项间蓝色分隔线
    .boblog-accordion-header 标题栏 可点击的标题栏，浅蓝背景，右侧箭头指示符
    .boblog-accordion-body 内容区 折叠项内容，默认隐藏，展开时显示
    .active .boblog-accordion-item 展开状态，标题栏变白底蓝字，箭头旋转，内容区显示
    .boblog-accordion-horizontal 容器 横向模式修饰类，各项水平排列，标题竖排
    .boblog-accordion-combined 容器 横竖汇合模式修饰类，顶层横排，子项内容区含嵌套竖向手风琴

    2. JavaScript API

    方法 参数 说明
    BoblogUI.accordion.init() 无 初始化页面内所有手风琴
    BoblogUI.accordion.init(container) HTMLElement 只初始化指定容器内的手风琴

    3. CSS 变量

    变量 默认值 说明
    --boblog-blue-wash #F4FBFF 标题栏默认背景色
    --boblog-border-blue #9EC9FF 容器边框、项间分隔线颜色
    --boblog-blue-accent #0D80BD 激活/悬停文字色
    --boblog-text #333 标题栏默认文字色
    --boblog-text-secondary #555 箭头指示符颜色

    4. data 属性

    属性 用在 说明
    data-bb-accordion-init .boblog-accordion JS 自动添加，标记已初始化，防止重复绑定

    基础用法（竖向）

    1. 规范说明

    • 容器类名: .boblog-accordion
    • 折叠项: .boblog-accordion-item（加 .active 为展开状态）
    • 标题栏: .boblog-accordion-header（可点击，带箭头指示符）
    • 内容区: .boblog-accordion-body（展开时显示）
    • 同一容器内同时只展开一个项（手风琴互斥模式）
    • 标题栏默认: 浅蓝背景 var(--boblog-blue-wash) + 蓝色边框
    • 标题栏激活/悬停: 白色背景 + 蓝色文字 var(--boblog-blue-accent)
    • 内容区: 白色背景 + 蓝色边框
    • 箭头: 默认朝右 ▶，展开时旋转朝下 ▼
    • 自动功能: DOMContentLoaded 自动绑定点击切换

    2. 演示

    第一节 - 什么是手风琴

    手风琴（Accordion）是一种常见的 UI 组件，用于在有限空间内展示多段内容。

    同一时间只展开一个面板，点击其他标题会自动收起当前面板。

    第二节 - 使用场景

    适用于 FAQ 页面、设置面板、分类导航等需要分组展示内容的场景。

    第三节 - 注意事项

    首项默认展开（加 active 类），也可以不设默认展开项。

    横向模式

    1. 规范说明

    • 容器加 .boblog-accordion-horizontal 修饰类
    • 各折叠项水平排列（flex 布局）
    • 标题栏在左侧竖排显示（writing-mode: vertical-lr）
    • 内容区在标题栏右侧展开
    • 切换逻辑与竖向相同（互斥模式）

    多项内容示例

    1. 规范说明

    • 支持任意数量的折叠项
    • 内容区可放任意 HTML（文本、列表、表格、代码块等）
    • 可以不设默认展开项（所有项均不加 active）

    2. 演示

    HTML 基础

    HTML（HyperText Markup Language）是构建网页的标准标记语言。

    • 定义页面结构和内容
    • 使用标签（tags）描述元素
    • 语义化标签提升可访问性
    CSS 样式

    CSS（Cascading Style Sheets）负责网页的视觉呈现。

    • 控制颜色、字体、间距、布局
    • 响应式设计适配多种屏幕
    • CSS 变量实现主题定制
    JavaScript 交互

    JavaScript 为网页添加交互行为和动态功能。

    浏览器兼容

    现代浏览器对 HTML5、CSS3、ES6+ 支持良好。

    开发工具

    推荐使用 VS Code、Chrome DevTools 等工具进行开发调试。

    横竖汇合模式

    1. 规范说明

    • 容器加 .boblog-accordion-combined 修饰类
    • 顶层横向排列（标题栏竖排），内容区嵌套竖向手风琴
    • 顶层各项互斥切换（点击分类标题切换）
    • 嵌套竖向手风琴各项也互斥切换（独立的手风琴实例）
    • 适用于分类导航、多级设置面板等场景

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描 .boblog-accordion
    • 无需手动调用任何函数
    • 已初始化的容器不会重复绑定（通过 data-bb-accordion-init 标记）

    2. 手动调用

    // 初始化页面所有手风琴
BoblogUI.accordion.init();

// 只初始化指定容器内的手风琴
BoblogUI.accordion.init(document.getElementById('myContainer'));

    3. 引入方式

    <!-- 引入 CSS（包含手风琴样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含手风琴逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/accordion.js"></script>
