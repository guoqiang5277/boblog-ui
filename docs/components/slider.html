    滑块 (Slider)

    基础用法

    • 基础类名: .boblog-slider（容器）
    • 输入元素: .boblog-slider-input（原生 <input type="range">）
    • 数值标签: .boblog-slider-value
    • 默认样式: 轨道高度 4px，手柄 14px×14px 方形，蓝色主题
    • 交互: 原生 range 元素，支持键盘和鼠标操作
    40 
    <div class="boblog-slider">
    <input type="range" class="boblog-slider-input" min="0" max="100" value="40">
    <span class="boblog-slider-value">40</span>
</div>

    竖向滑块

    • 容器类名: .boblog-slider-vertical（独立容器，非 .boblog-slider 修饰类）
    • 实现方式: div 模拟 + JS 拖拽交互（非原生 range）
    • 轨道: .boblog-slider-vertical-track（4px 宽竖条）
    • 手柄: .boblog-slider-vertical-thumb（14x14 方形，蓝色边框）
    • 配置属性: data-min / data-max / data-value / data-step
    • 交互: 鼠标拖拽手柄、点击轨道跳转、触屏滑动
    • 颜色变体: .success / .warning / .danger

    滑块 JavaScript API

    1. 自动初始化

    页面加载后自动初始化所有 .boblog-slider-vertical 组件。JS 负责：手柄拖拽、轨道点击跳转、触屏滑动、实时更新数值标签。水平滑块使用原生 <input type="range">，无需 JS。

    2. 手动调用

    // 初始化指定容器内的竖向滑块（用于动态插入的内容）
BoblogUI.slider.init(document.getElementById('myContainer'));

    3. 引入方式

    <link rel="stylesheet" href="../boblog-ui.css">
<script src="../boblog-ui.js"></script>
