    对话框

    基础能力

    • 统一入口: BoblogUI.Dialog.alert / confirm / prompt
    • 底层实现: 复用 BoblogUI.Modal 与现有 button / input / message 样式
    • 宿主策略: 页面首次调用时自动创建全局唯一宿主，不要求业务项目手写固定 DOM
    • 交互约定: ESC、右上角关闭、点击遮罩都会返回受控结果，而不是静默丢失状态
    • 声明式确认: 支持 data-boblog-confirm 直接绑定 form / a / button
    点击按钮查看返回结果。

    JavaScript API

    1. alert

    await BoblogUI.Dialog.alert('保存成功', {
    title: '提示',
    confirmText: '我知道了',
    closeOnMask: true,
    closeOnEsc: true
});

    返回值：Promise<void>

    2. confirm

    var ok = await BoblogUI.Dialog.confirm('确认删除此记录？', {
    title: '删除确认',
    confirmText: '确认删除',
    cancelText: '取消',
    danger: true,
    closeOnMask: true,
    closeOnEsc: true
});

    返回值：Promise<boolean>。点击确认返回 true，关闭/取消返回 false。

    3. prompt

    var value = await BoblogUI.Dialog.prompt('请输入拒绝原因', {
    title: '拒绝退款',
    placeholder: '请输入拒绝原因',
    defaultValue: '',
    required: true,
    errorMessage: '拒绝原因不能为空',
    confirmText: '确认',
    cancelText: '取消',
    danger: true
});

    返回值：Promise<string | null>。点击确认并校验通过后返回去首尾空格的字符串，取消/关闭返回 null。

    4. 声明式确认
