    评分 (Rating)

    基础用法

    核心类名

    • .boblog-rating - 评分容器
    • .boblog-rating-star - 星星元素
    • .boblog-rating-star.active - 激活的星星
    • .boblog-rating-text - 评分文本

    <span class="boblog-rating-star">★</span>
    <span class="boblog-rating-star">★</span>
    <span class="boblog-rating-text">3.0 分</span>
</div>

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描 .boblog-rating:not(.readonly):not(.disabled)
    • 无需手动调用任何函数

    2. 手动调用

    // 初始化页面所有评分组件
BoblogUI.rating.init();

// 只初始化指定容器内的评分组件
BoblogUI.rating.init(document.getElementById('myContainer'));

    3. 引入方式

    <!-- 引入 CSS（包含评分组件样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含评分组件逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/rating.js"></script>
