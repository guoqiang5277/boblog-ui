    基础用法

    1. 规范说明

    • 容器类名: .boblog-codeblock
    • 背景色: var(--boblog-white) / #FFFFFF
    • 左侧蓝色指示条: 3px solid var(--boblog-blue)
    • 边框: 1px solid var(--boblog-border)
    • 内边距: 16px 20px
    • 代码字体: Consolas / Monaco / Courier New, 13px
    • 行高: 1.7
    • 语法高亮: Prism.js v1.29.0（支持 17 种语言）
    • 语言指定: 在 <code> 上添加 class="language-xxx"
    • 自动功能: 行号显示、复制按钮、语法高亮

    可选属性 (data-*)

    1. 规范说明

    • data-no-line-numbers — 禁用行号显示
    • data-no-copy — 禁用复制按钮
    • data-no-highlight — 禁用语法高亮

    语法高亮（Prism.js）

    1. 规范说明

    • 语法高亮引擎: Prism.js v1.29.0
    • 语言指定: 在 <code> 上添加 class="language-xxx"
    • 支持语言: HTML、CSS、JavaScript、PHP、Python、C#、Java、Go、SQL、Bash、JSON、TypeScript、Ruby、Rust、YAML、Markdown、Docker
    • 注释 — .token.comment: #6A8A6A 灰绿斜体
    • 关键字 — .token.keyword: var(--boblog-blue) 品牌蓝加粗
    • 属性名/选择器 — .token.selector/.attr-name: #9C3328 深红棕
    • 字符串 — .token.string: #2E7D32 棕绿
    • 标签名/函数 — .token.tag/.function: #1A6B8A 深蓝
    • 数字/布尔值 — .token.number/.boolean: #7B5EA7 紫色
    • 标点 { } : ; — .token.punctuation: var(--boblog-text-muted) 灰色

    2. CSS 高亮演示

    4. JavaScript 高亮演示

    /**
 * 初始化代码块组件
 * @param {Element} container - 搜索范围
 */
function initCodeblock(container) {
    var blocks = document.querySelectorAll('.boblog-codeblock');
    var count = blocks.length;

    blocks.forEach(function (block, index) {
        if (block.hasAttribute('data-no-highlight')) return;

        var code = block.querySelector('code');
        if (code && typeof Prism !== 'undefined') {
            Prism.highlightElement(code);
        }
        console.log('已处理第 ' + (index + 1) + '/' + count + ' 个代码块');
    });

    return true;
}

    复制按钮

    1. 规范说明

    • 按钮类名: .boblog-copy-btn（由 JS 自动注入）
    • 位置: 代码块右上角 (absolute top:8px right:8px)
    • 默认状态: 浅蓝背景 + 蓝色边框文字
    • 悬停状态: 蓝色背景 + 白色文字
    • 复制成功: 绿色边框文字 + 浅绿背景，1.5 秒后恢复
    • 复制内容为纯文本（去除 HTML 标签和行号）

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描 .boblog-codeblock
    • 无需手动调用任何函数

    2. 手动调用

    // 初始化页面所有代码块
BoblogUI.codeblock.init();

// 只初始化指定容器内的代码块
BoblogUI.codeblock.init(document.getElementById('myContainer'));

// 单独调用某个功能
BoblogUI.codeblock.highlightAll();     // 语法高亮
BoblogUI.codeblock.addLineNumbers();   // 行号
BoblogUI.codeblock.addCopyButtons();   // 复制按钮

    3. 引入方式

    <!-- 引入 CSS（包含代码块样式 + Prism token 配色） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（已集成 Prism.js 语法高亮引擎 + 代码块逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/vendor/prism/prism.min.js"></script>
<script src="src/js/codeblock.js"></script>
    • boblog-ui.js 已内置 Prism.js v1.29.0（含 17 种语言），无需额外引入
    • boblog-ui.css 已包含 Prism token 的 Bo-Blog 配色映射
