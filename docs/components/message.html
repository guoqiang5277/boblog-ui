    消息提示 (Message)

    基础用法

    • 结构: .boblog-message > .boblog-message-icon + .boblog-message-content
    • 布局: flex 水平排列，图标 + 内容
    • 内边距: 10px 15px (--boblog-spacing-sm / --boblog-spacing-md)
    • 边框: 1px solid #9EC9FF (--boblog-border-blue)
    • 背景色: #F5F8FB (--boblog-bg-subtle)
    • 字体大小: 12px (--boblog-font-size-sm)
    • 行高: 1.5
    • 底部间距: 15px (--boblog-spacing-md)
    ℹ 这是一条默认消息提示。 
    <div class="boblog-message">
    <span class="boblog-message-icon">ℹ</span>
    <span class="boblog-message-content">这是一条默认消息提示。</span>
</div>

    四种消息类型

    • .boblog-message-success: 成功消息，边框 #3C8C30 (--boblog-success)，背景 #F0F8EE (--boblog-success-bg)
    • .boblog-message-error: 错误消息，边框 #CC3333 (--boblog-danger)，背景 #FFF0F0 (--boblog-danger-bg)
    • .boblog-message-warning: 警告消息，边框 #D07020 (--boblog-orange)，背景 #FFF8E6 (--boblog-warning-bg)
    • .boblog-message-info: 信息消息，边框 #1677B5 (--boblog-info)，背景 #EDF5FB (--boblog-info-bg)
    ✓ 操作成功！数据已保存。
    ✕ 操作失败！请检查输入信息。
    ⚠ 警告：此操作将影响系统配置。
    ℹ 提示：请先完成基本设置。

    可关闭消息

    • 结构: .boblog-message.boblog-message-dismissible > .boblog-message-icon + .boblog-message-content + button.boblog-message-close
    • 右内边距: 30px (--boblog-spacing-2xl)，为关闭按钮留空间
    • 关闭按钮: 绝对定位，右侧 10px，垂直居中
    • 关闭按钮颜色: #888888 (--boblog-text-muted)，hover 变为 #1A1A1A (--boblog-text)
    ✕ 操作失败！请检查输入信息。
    ⚠ 警告：此操作将影响系统配置。 
    <!-- 可关闭的错误消息 -->
<div class="boblog-message boblog-message-error boblog-message-dismissible">
    <span class="boblog-message-icon">✕</span>
    <span class="boblog-message-content">操作失败！请检查输入信息。</span>
    <button class="boblog-message-close" onclick="this.parentElement.style.display='none'">×</button>
</div>

<!-- 可关闭的警告消息 -->
<div class="boblog-message boblog-message-warning boblog-message-dismissible">
    <span class="boblog-message-icon">⚠</span>

</div>

    紧凑模式

    • 类名: .boblog-message-compact
    • 内边距: 5px 10px (--boblog-spacing-xs / --boblog-spacing-sm)
    • 字体大小: 11px (--boblog-font-size-xs)
    • 适合空间受限场景，如表格内、侧边栏内等

    居中对齐

    • 类名: .boblog-message-center
    • 对齐方式: justify-content: center + text-align: center
    • 适合单行简短提示，内容水平居中显示
    ℹ 居中显示的信息提示
    ✓ 操作完成
