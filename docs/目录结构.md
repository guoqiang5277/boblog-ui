# boblog-ui 目录结构规范

## src 目录结构

```
src/
├── base/           # 基础
├── controls/       # 基础控件
├── components/     # 复合组件
├── layout/         # 布局
├── frontend/       # 前台专用
└── js/             # JavaScript
```

## 分类说明

### base/ — 基础
1. variables — CSS 变量
2. colors — 颜色
3. fonts — 字体定义
4. typography — 排版
5. spacing — 间距/尺寸

### controls/ — 基础控件
6. button — 按钮
7. input — 输入框
8. checkbox — 复选框
9. radio — 单选框
10. textarea — 文本域
11. select — 下拉选择
12. progress-bar — 进度条
13. image — 图片/头像/缩略图/画廊
14. border — 边框工具类

### components/ — 复合组件

**数据录入：**
15. editor — 富文本编辑器
16. date-picker — 日期选择器
17. rating — 评分
18. slider — 滑块

**数据展示：**
19. table — 表格
20. tree — 树形控件
21. badge — 徽章
22. stat-card — 统计卡片
23. timeline — 时间线
24. codeblock — 代码块
25. blockquote — 引用块

**导航：**
26. tabs — 标签页
27. toc — 目录导航
28. pagination — 分页
29. breadcrumb — 面包屑

**反馈：**
30. modal — 模态框
31. message — 消息提示
32. empty-state — 空状态

**容器：**
33. panel — 面板

### layout/ — 布局
34. grid — 栅格系统
35. page-structure — 页面结构
36. responsive — 响应式
37. form-layout — 表单布局
38. doc-layout — 文档布局
39. nav-sidebar — 侧边栏导航
40. nav-frontend — 前台导航

### frontend/ — 前台专用
41. date-card — 日期卡片
42. sidebar-panel — 侧边栏面板
43. article-list — 文章列表
44. comment — 评论

## docs 目录结构

```
docs/
├── index.html      # 主页（项目介绍、快速开始、版本信息）
├── base/           # 基础文档
├── controls/       # 基础控件文档
├── components/     # 复合组件文档
├── layout/         # 布局文档
└── frontend/       # 前台文档
```
