项目介绍

boblog-ui 是基于 Bo-Blog V2.1 Default Skin 的经典博客 UI 组件库，完美还原 2000 年代独立博客的视觉风格。

  • 蓝白色调，清爽简洁，回归经典博客审美
  • 无圆角设计（方角），所有边框保持直角
  • Tahoma, Arial 字体优先，12px 基础字号，150% 行高
  • 类名前缀统一使用 boblog-
  • 纯 CSS 实现，部分组件提供 JS 增强
快速开始

1. 引入 CSS：

<link rel="stylesheet" href="boblog-ui.css">

2. 引入 JS（可选，用于交互组件）：

<script src="boblog-ui.js"></script>

3. 使用组件：

<button class="boblog-btn boblog-btn-primary">按钮</button>
