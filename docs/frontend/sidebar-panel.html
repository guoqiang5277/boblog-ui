    侧边栏面板 (Sidebar Panel)

    列表面板 (.boblog-sidebar-panel)

    • 面板容器: .boblog-sidebar-panel，margin-bottom 15px，边框 1px solid #9EC9FF，白色背景
    • 面板标题 (.boblog-sidebar-title): font-size 14px (--boblog-font-size-md)，font-weight bold，颜色 #333，背景 #F5FBFE (--boblog-bg-panel)，底边框 1px solid #9EC9FF，padding 8px 10px
    • 面板内容 (.boblog-sidebar-content): padding 10px
    • 列表 (.boblog-sidebar-list): 无列表样式，无 margin/padding
    • 列表项 (.boblog-sidebar-list-item): padding 5px 0，底部虚线分隔 1px dashed #D3E7FC，12px，最后一项无底边框
    • 列表项链接: 颜色 #333，hover 时 #0D80BD，单行溢出省略号
    • 计数 (.boblog-sidebar-list-count): 右浮动，灰色 #999，11px
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    使用说明

    适用场景

    • 博客侧边栏内容展示
    • 最新文章、热门文章列表
    • 分类目录、标签云
    • 友情链接、站点信息
    • 搜索框、RSS 订阅

    设计要点

    • 面板标题使用浅蓝色背景 #F5FBFE，与白色内容区对比
    • 边框使用浅蓝色 #9EC9FF，保持整体色调一致
    • 列表项单行溢出省略号，适配窄侧边栏
    • 标签云采用 flex 流式布局，自动换行
    • 无圆角、无阴影，保持 2000 年代经典风格
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    多面板布局

    • 面板间距: margin-bottom 15px（自带）
    • 右侧侧边栏宽度: 201px 或 140px（由 page-structure 布局控制）
    • 面板宽度自适应侧边栏宽度
