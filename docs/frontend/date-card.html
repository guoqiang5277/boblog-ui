    日期标签卡片 (Date Card)

    基础日期卡片 (.boblog-date-card)

    • 容器: .boblog-date-card，宽 42px
    • 月份区 (.boblog-date-month): 背景 #2E93C9，白色文字
    • 日期区 (.boblog-date-day): 背景 #EFF6FE，深色文字
    • 年份区 (.boblog-date-year): 可选，显示在日期区下方
    • 无圆角设计，保持 2000 年代经典风格
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    使用说明

    适用场景

    • 文章列表左侧日期标签
    • 时间轴视图
    • 归档页面日期展示

    设计要点

    • 月份使用深蓝色背景 #2E93C9，增强视觉识别
    • 日期使用浅蓝色背景 #EFF6FE，与月份形成对比
    • 无圆角设计，保持 2000 年代经典风格
    • 年份为可选元素，适用于跨年归档场景
    • 本组件为纯 CSS 实现，无需引入 JavaScript
