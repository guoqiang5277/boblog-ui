    文章列表 (Article List)

    文章列表 (.boblog-article-list)

    • 列表容器: .boblog-article-list，width 100%
    • 文章项 (.boblog-article-item): display flex，padding 15px 0，底部虚线分隔 1px dashed #D3E7FC (--boblog-blue-border-list)，最后一项无底边框
    • 日期区 (.boblog-article-date): flex-shrink 0，右间距 15px，内嵌日期卡片 .boblog-date-card
    • 内容区 (.boblog-article-content): flex 1，min-width 0
    • 标题 (.boblog-article-title): font-size 16px (--boblog-font-size-lg)，font-weight bold，链接颜色 #0D80BD (--boblog-blue-primary)
    • 元信息 (.boblog-article-meta): font-size 12px (--boblog-font-size-sm)，颜色 #999 (--boblog-text-gray)，各项用竖线 | 分隔
    • 摘要 (.boblog-article-summary): font-size 12px，颜色 #333 (--boblog-text-dark)，行高 1.8
    • 标签 (.boblog-article-tags + .boblog-article-tag): 背景 #F4FBFF，边框 1px solid #9EC9FF，文字 #0D80BD，11px
    • 本组件为纯 CSS 实现，无需引入 JavaScript

    组件结构

    布局说明

    • 文章项: display flex 横向排列，日期卡片在左，内容区在右
    • 左侧日期卡片: flex-shrink 0 固定不缩放，右间距 15px
    • 右侧内容区: flex 1 自适应宽度，min-width 0 防止溢出
    • 文章项分隔: 底部虚线 1px dashed #D3E7FC，最后一项无底边框

    元信息说明

    • 作者、分类、浏览量、评论数等信息
    • 使用 CSS 伪元素 ::after 自动添加竖线 | 分隔各项
    • 分类链接继承默认链接颜色

    标签设计

    • 背景色: #F4FBFF (--boblog-bg-content)
    • 边框色: #9EC9FF (--boblog-blue-border)
    • 文字色: #0D80BD (--boblog-blue-primary)
    • 无圆角，保持经典风格

    使用说明

    适用场景

    • 博客首页文章列表
    • 分类页面文章列表
    • 标签页面文章列表
    • 搜索结果页面

    设计要点

    • 左侧日期卡片与内容区对齐，增强时间感
    • 标题使用 16px 加粗，突出重点
    • 摘要文字保持 12px，行高 1.8，易于阅读
    • 标签紧贴摘要，方便快速识别文章主题
    • 本组件为纯 CSS 实现，无需引入 JavaScript
