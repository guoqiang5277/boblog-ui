    文本域 (Textarea)

    文本域 (.boblog-textarea)

    • 边框: 1px solid #DDE8F0 (--boblog-border-light)
    • 内边距: 6px 10px
    • 字体大小: 12px (--boblog-font-size-sm)
    • 行高: 1.5
    • 背景色: #FFFFFF (--boblog-bg)
    • 文字颜色: #1A1A1A (--boblog-text)
    • 最小高度: 80px
    • 默认 resize: vertical（仅垂直调整）
    • Hover: 边框 #C4D7E8 (--boblog-border)
    • Focus: 边框 #0D80BD (--boblog-blue-accent)，背景 #E8F0F7 (--boblog-blue-pale)，outline 2px solid rgba(13, 128, 189, 0.1)
    • Placeholder: 颜色 #888888 (--boblog-text-muted)，opacity 0.8
    • 基础样式为纯 CSS 实现；字符计数功能需引入 boblog-ui.js（详见 字符计数 及 JavaScript API 章节）
    <textarea class="boblog-textarea" rows="4" placeholder="请输入内容..."></textarea>

    状态变体

    • 禁用 (disabled): 背景 #F5F5F5 (--boblog-disabled-bg)，边框 #D9D9D9 (--boblog-disabled-border)，文字 #888888，cursor not-allowed，resize none
    • 只读 (read-only): 背景 #F5F8FB (--boblog-bg-subtle)，cursor default，resize none
    • 错误 (.boblog-textarea-error 或 .error): 边框 #CC3333 (--boblog-danger)，focus 时 outline rgba(224, 62, 45, 0.1)
    <!-- 禁用状态 -->
<textarea class="boblog-textarea" rows="3" disabled></textarea>

<!-- 只读状态 -->
<textarea class="boblog-textarea" rows="3" readonly>只读内容</textarea>

    调整大小 (Resize)

    • 默认: resize vertical（仅垂直拖拽）
    • .boblog-textarea-fixed: resize none（不可调整）
    • .boblog-textarea-horizontal: resize horizontal（仅水平拖拽）
    • .boblog-textarea-both: resize both（自由拖拽）
    <!-- 不可调整大小 -->
<textarea class="boblog-textarea boblog-textarea-fixed" rows="3"></textarea>

<!-- 仅垂直调整（默认） -->
<textarea class="boblog-textarea" rows="3"></textarea>

<!-- 仅水平调整 -->

<!-- 自由调整 -->

    字符计数

    • 结构: .boblog-textarea-wrapper > .boblog-textarea[data-maxlength] + .boblog-textarea-count
    • 上限属性: data-maxlength="N"，未设置则不启用计数
    • 计数位置: 右下角绝对定位，12px，半透明白底
    • 超限样式: 计数红色加粗，边框变红，聚焦背景 #FFF0F0
    • 需引入 boblog-ui.js（详见 JavaScript API）
    <!-- 字符计数（200字上限） -->
<div class="boblog-textarea-wrapper">
    <textarea class="boblog-textarea" rows="4"
              data-maxlength="200"
              placeholder="请输入内容（最多200字）..."></textarea>
</div>

<!-- 字符计数（20字上限，预填充内容已接近上限） -->
<div class="boblog-textarea-wrapper">
    <textarea class="boblog-textarea" rows="4"
              data-maxlength="20">这是一段已经接近字数上限的示例文本内容，请试试继续输入</textarea>
</div>

    JavaScript API

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描所有 .boblog-textarea-wrapper
    • 自动为含 data-maxlength 属性的 textarea 启用字符计数
    • 无需手动调用任何函数

    2. 手动调用

    // 初始化页面所有文本域计数
BoblogUI.textarea.init();

// 只初始化指定容器内的文本域计数
BoblogUI.textarea.init(document.getElementById('myContainer'));

    3. 引入方式

    <!-- 引入 CSS（包含文本域样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含字符计数逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/textarea.js"></script>
