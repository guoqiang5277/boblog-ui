    输入框 (Input)

    输入框 (.boblog-input)

    • 边框: 1px solid #DDE8F0 (--boblog-border-light)
    • 内边距: 6px 10px
    • 字体大小: 12px (--boblog-font-size-sm)
    • 背景色: #FFFFFF (--boblog-bg)
    • 文字颜色: #1A1A1A (--boblog-text)
    • Hover 状态: 边框 #C4D7E8 (--boblog-border)
    • Focus 状态: 边框 #0D6EAA (--boblog-blue)，背景 #E8F0F7 (--boblog-blue-pale)，外框 2px solid rgba(13,128,189,0.1)
    • Disabled 状态: 背景 #F5F5F5，边框 #D9D9D9，文字 #888888 (--boblog-text-muted)，光标 not-allowed
    • Error 状态 (.boblog-input-error): 边框 #CC3333 (--boblog-danger)
    <!-- 默认状态 -->
<input type="text" class="boblog-input" placeholder="默认状态" style="width: 200px;">

<!-- Focus 状态 (请点击) -->
<input type="text" class="boblog-input" value="Focus 状态 (请点击)" style="width: 200px;">

<!-- 禁用状态 -->
<input type="text" class="boblog-input" placeholder="禁用状态" disabled style="width: 200px;">

<!-- 错误状态 -->
<input type="text" class="boblog-input boblog-input-error" placeholder="错误状态" style="width: 200px;">

    输入框组 (.boblog-input-group)

    • 容器: display flex
    • 前缀/后缀插件 (.boblog-input-group-addon): 背景 #F5F8FB (--boblog-bg-subtle)，边框 1px solid #DDE8F0 (--boblog-border-light)，内边距 6px 10px
    • 输入框与插件共享边框，中间无间隙
    • 支持左侧前缀、右侧后缀或按钮组合
    https://
    ¥ .00 
    <!-- 前缀插件 -->
<div class="boblog-input-group" style="width: 300px;">

    <input type="text" class="boblog-input" placeholder="your-domain.com">

    密码输入框 (.boblog-input-password)

    • 容器: position relative，display inline-block，width 100%
    • 输入框右侧留 36px 空间给切换按钮（padding-right: 36px）
    • 切换按钮 (.boblog-input-password-toggle): 绝对定位右侧，宽 32px，背景透明，颜色 #888888 (--boblog-text-muted)，hover 时 #1A1A1A (--boblog-text)
    • 默认隐藏密码: 不加 data-visible 或 data-visible="false"
    • 默认显示密码: data-visible="true"（input type 自动设为 text）
    • 隐藏切换按钮: data-toggle-password="false"（纯密码输入框，无眼睛按钮）
    • 显示切换按钮: 不加 data-toggle-password 或 data-toggle-password="true"（默认）
    • 禁用状态: 切换按钮 cursor not-allowed，颜色 #AAAAAA (--boblog-text-disabled)
    • 依赖 boblog-ui.js（BoblogUI.input 模块）

    密码输入框 - JavaScript API

    1. 自动初始化

    引入 boblog-ui.js 后，页面加载完成时会自动扫描所有 .boblog-input-password 容器并初始化切换按钮，无需手动调用。

    2. 手动调用

    // 初始化指定容器内的密码输入框
BoblogUI.input.init(document.getElementById('myForm'));

// 初始化整个页面（等同于自动初始化）
BoblogUI.input.init();

    3. 引入方式

    <!-- CSS -->
<link rel="stylesheet" href="../boblog-ui.css">

<!-- JS（密码切换功能需要） -->
<script src="../boblog-ui.js"></script>
