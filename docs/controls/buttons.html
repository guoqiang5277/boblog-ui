    按钮组件 (Buttons)

    基础按钮 (.boblog-btn)

    • 背景色: #FFFFFF (--boblog-bg)
    • 边框: 1px solid #C4D7E8 (--boblog-border)
    • 文字颜色: #1A1A1A (--boblog-text)
    • 内边距: 6px 15px
    • 字体大小: 12px (--boblog-font-size-sm)
    • 鼠标悬停: 边框/文字 #0D6EAA (--boblog-blue)，背景 #E8F0F7 (--boblog-blue-pale)
    • 鼠标按下: 背景色 #e6f4fb
    <button class="boblog-btn">基础按钮</button>
<button class="boblog-btn" disabled>禁用状态</button>
    <!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>基础按钮示例 - Bo-Blog UI</title>
    <link rel="stylesheet" href="../../dist/boblog-ui.css">
</head>
<body>
    <button class="boblog-btn">基础按钮</button>
    <button class="boblog-btn" disabled>禁用状态</button>
</body>
</html>

    尺寸变体

    • 小按钮 (.boblog-btn-sm): 内边距 4px 10px, 字体 12px
    • 标准按钮: 内边距 6px 15px, 字体 12px
    • 大按钮 (.boblog-btn-lg): 内边距 8px 20px, 字体 14px

    按钮组 (.boblog-btn-group)

    • 容器: display inline-flex
    • 按钮间无间距, 共享边框
    • 第一个按钮: 左侧圆角
    • 最后一个按钮: 右侧圆角
    • 中间按钮: 无圆角
    • 相邻按钮边框合并避免双线
    <div class="boblog-btn-group">
    <button class="boblog-btn boblog-btn-primary">左</button>
    <button class="boblog-btn boblog-btn-primary">中</button>
    <button class="boblog-btn boblog-btn-primary">右</button>
</div>

<div class="boblog-btn-group">
    <button class="boblog-btn">列表</button>
    <button class="boblog-btn">网格</button>
    <button class="boblog-btn">卡片</button>
</div>

    组合示例

    <!-- 确认操作 -->

<button class="boblog-btn">取消</button>

<button class="boblog-btn boblog-btn-secondary boblog-btn-sm">重置</button>
<button class="boblog-btn boblog-btn-danger boblog-btn-sm">删除</button>

<!-- 次要操作 -->
