    下拉选择 (Select)

    下拉选择 (.boblog-select)

    • 结构: .boblog-select（容器）> select（原生下拉元素）
    • 依赖: boblog-ui.css + boblog-ui.js（JS 自动将原生 select 转换为自定义下拉面板）
    • 边框: 1px solid #DDE8F0 (--boblog-border-light)
    • 内边距: 6px 30px 6px 10px（右侧留空放自定义箭头）
    • 字体大小: 12px (--boblog-font-size-sm)
    • 背景色: #FFFFFF (--boblog-bg)
    • 文字颜色: #1A1A1A (--boblog-text)
    • 自定义箭头: CSS 三角形，颜色 #888888 (--boblog-text-muted)，展开时旋转 180°
    • Hover 状态: 边框 #C4D7E8 (--boblog-border)
    • Focus/展开 状态: 边框 #0D6EAA (--boblog-blue)，背景 #E8F0F7 (--boblog-blue-pale)
    • 选项悬停: 背景 #E8F0F7 (--boblog-blue-pale)，文字 #0D6EAA (--boblog-blue)
    • 选中项: 背景 #0D6EAA (--boblog-blue)，文字 #FFFFFF
    • Disabled 状态: 背景 #F5F5F5，边框 #D9D9D9，文字 #888888，箭头 #D9D9D9
    • Error 状态 (.boblog-select-error): 边框 #CC3333 (--boblog-danger)
    <!-- 默认状态 -->
<div class="boblog-select">
    <select>
        <option>请选择...</option>
        <option>选项 1</option>
        <option>选项 2</option>
        <option>选项 3</option>
    </select>
</div>

    </select>

        <option>选项 A</option>
    </select>

    多选下拉框

    • 给 <select> 添加 multiple 属性启用多选模式
    • 多选模式下不显示下拉箭头
    • 内边距: 5px 10px，高度自适应
    • 注意：多选模式使用原生渲染，不会被 JS 转换为自定义下拉
    <div class="boblog-select">
    <select multiple>
        <option>选项 1</option>
        <option>选项 2</option>
        <option>选项 3</option>
        <option>选项 4</option>
        <option>选项 5</option>
    </select>
</div>

    JavaScript API

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描 .boblog-select 并转换为自定义下拉
    • 多选下拉框（select[multiple]）不会被转换，保留原生渲染
    • 转换后原生 <select> 隐藏但保留在 DOM 中，表单提交不受影响
    • 支持键盘操作：↑↓ 切换选项、Enter/空格 确认、Escape 关闭
    • 点击页面其他区域自动关闭下拉面板

    1. 自动初始化

    • 引入 boblog-ui.js 后，DOMContentLoaded 自动扫描所有 .boblog-select
    • 无需手动调用任何函数

    2. 手动调用

    // 初始化页面所有下拉选择
BoblogUI.select.init();

// 只初始化指定容器内的下拉选择
BoblogUI.select.init(document.getElementById('myForm'));

    3. 引入方式

    <!-- 引入 CSS（包含下拉选择样式） -->
<link rel="stylesheet" href="dist/boblog-ui.css">

<!-- 引入 JS（包含下拉选择逻辑） -->
<script src="dist/boblog-ui.js"></script>

<!-- 或开发时单独引入 -->
<script src="src/js/select.js"></script>
