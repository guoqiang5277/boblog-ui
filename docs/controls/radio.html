    单选框 (Radio)

    单选框 (.boblog-radio)

    • 容器: display inline-flex，align-items center，cursor pointer
    • 原生 radio: 通过 .boblog-radio-input 隐藏（opacity: 0），使用伪元素绘制圆形单选框
    • 未选中: 边框 1px solid #C4D7E8 (--boblog-border)，背景 #FFFFFF (--boblog-bg)，圆形（border-radius: 50%）
    • 已选中: 边框 #0D6EAA (--boblog-blue)，背景 #0D6EAA (--boblog-blue)，内圆点 #FFFFFF 8px
    • Hover: 边框变为 #0D6EAA (--boblog-blue)
    • Focus: outline 2px solid rgba(13, 128, 189, 0.3)，offset 2px
    • Disabled 未选中: 背景 #F5F5F5，边框 #D9D9D9，文字 #888888 (--boblog-text-muted)
    • Disabled 已选中: 背景 #7FBFDD，边框 #7FBFDD，内圆点 #FFFFFF
    • 标签间距: padding-left 24px（含 16px 单选框 + 间距）
    • 本组件为纯 CSS 实现，无需引入 JavaScript
    <!-- 正常状态 -->
<label class="boblog-radio">
    <input type="radio" class="boblog-radio-input" name="group1">
    <span class="boblog-radio-label">选项 1</span>
</label>
<label class="boblog-radio">
    <input type="radio" class="boblog-radio-input" name="group1" checked>
    <span class="boblog-radio-label">选项 2</span>
</label>
<label class="boblog-radio">
    <input type="radio" class="boblog-radio-input" name="group1">
    <span class="boblog-radio-label">选项 3</span>
</label>

<!-- 禁用状态 -->
<label class="boblog-radio">
    <input type="radio" class="boblog-radio-input" name="group2" disabled>
    <span class="boblog-radio-label">禁用未选中</span>
</label>
<label class="boblog-radio">
    <input type="radio" class="boblog-radio-input" name="group2" checked disabled>
    <span class="boblog-radio-label">禁用已选中</span>
</label>

    单选框组 (.boblog-radio-group)

    • 垂直排列（默认）: flex-direction column，间距 10px (--boblog-spacing-sm)
    • 水平排列 (.inline): flex-direction row，flex-wrap wrap，间距 15px (--boblog-spacing-md)
    • 无标签文本: .boblog-radio-label:empty 时 padding-left 缩至 16px

    垂直排列（默认）

    水平排列 (.inline)

    <!-- 垂直排列（默认） -->
<div class="boblog-radio-group">
