    复选框 (Checkbox)

    复选框 (.boblog-checkbox)

    • 容器: display inline-flex，align-items center，cursor pointer
    • 原生 checkbox: 通过 .boblog-checkbox-input 隐藏（opacity: 0），使用伪元素绘制方形复选框
    • 未选中: 边框 1px solid #C4D7E8 (--boblog-border)，背景 #FFFFFF (--boblog-bg)
    • 已选中: 边框 #0D6EAA (--boblog-blue)，背景 #0D6EAA (--boblog-blue)，勾选标记 #FFFFFF
    • Hover: 边框变为 #0D6EAA (--boblog-blue)
    • Focus: outline 2px solid rgba(13, 128, 189, 0.3)，offset 2px
    • Disabled 未选中: 背景 #F5F5F5，边框 #D9D9D9，文字 #888888 (--boblog-text-muted)
    • Disabled 已选中: 背景 #7FBFDD，边框 #7FBFDD
    • 标签间距: padding-left 24px（含 16px 复选框 + 间距）
    • 本组件为纯 CSS 实现，无需引入 JavaScript
    <!-- 未选中 -->
<label class="boblog-checkbox">
    <input type="checkbox" class="boblog-checkbox-input">
    <span class="boblog-checkbox-label">未选中</span>
</label>

<!-- 已选中 -->
<label class="boblog-checkbox">
    <input type="checkbox" class="boblog-checkbox-input" checked>
    <span class="boblog-checkbox-label">已选中</span>
</label>

<!-- 禁用未选中 -->
<label class="boblog-checkbox">
    <input type="checkbox" class="boblog-checkbox-input" disabled>
    <span class="boblog-checkbox-label">禁用未选中</span>
</label>

<!-- 禁用已选中 -->
<label class="boblog-checkbox">
    <input type="checkbox" class="boblog-checkbox-input" checked disabled>
    <span class="boblog-checkbox-label">禁用已选中</span>
</label>

    半选状态 (Indeterminate)

    • 半选状态: 背景 #0D6EAA (--boblog-blue)，边框 #0D6EAA (--boblog-blue)
    • 标记: 水平短横线（非勾选），宽 8px，白色 2px 底边框
    • 半选状态需通过 JavaScript 设置 checkbox.indeterminate = true，无法通过 HTML 属性设置
    • 常用于"全选"场景：部分子项选中时，父级复选框显示半选状态
    <label class="boblog-checkbox">
    <input type="checkbox" class="boblog-checkbox-input" id="indeterminateDemo">
    <span class="boblog-checkbox-label">半选状态</span>
</label>

<script>
// 半选状态只能通过 JS 设置
document.getElementById('indeterminateDemo').indeterminate = true;
</script>

    复选框组 (.boblog-checkbox-group)

    • 垂直排列（默认）: flex-direction column，间距 10px (--boblog-spacing-sm)
    • 水平排列 (.inline): flex-direction row，flex-wrap wrap，间距 15px (--boblog-spacing-md)
    • 无标签文本: .boblog-checkbox-label:empty 时 padding-left 缩至 16px

    垂直排列（默认）

    水平排列 (.inline)
